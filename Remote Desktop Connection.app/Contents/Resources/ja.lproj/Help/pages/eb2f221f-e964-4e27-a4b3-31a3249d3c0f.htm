ドキュメントをプリントする

すべて表示

Windows ベースのコンピューターにあるドキュメントを、Mac に接続されているプリンターでプリントできます。そのためには、まずプリンターを Windows ベースのコンピューターと共有する必要があります。

プリンターを利用できるようにすると、Windows ベースのコンピューターからプリンターにアクセスできます。接続ファイルリモート接続に関するすべての情報を格納するファイル。たとえば、セッション ウインドウの画面サイズや Windows ベースのコンピューターから使用できるようにする Mac ドライブなどの設定は、接続ファイルに保存できます。Windows ベースの複数のコンピューターに接続する場合は、接続するコンピューターごとに接続ファイルを保存することができます。を保存しておくと、後のリモート セッションでその接続ファイルを使ってログインしたときに、プリンターはそのまま利用できます。

必要に応じて、次の操作を行います。

プリンターを Windows ベースのコンピューターから利用できるようにする

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [プリンター] をクリックし、 [Mac に接続されたプリンターを使用する] チェック ボックスをオンにします。

  3. 目的のプリンターを選択します。

    ヒント 
     

    • プリント時にプリンターを選択する場合は、 [すべてのプリンター] をクリックします。

    • プリンターへのアクセス情報を含む接続ファイルを保存するには、 [ファイル] メニューの [別名で保存] をクリックし、接続ファイル名を入力します。

Windows ベースのドキュメントを Mac に接続されているプリンターでプリントする

  1. Windows デスクトップで、プリントするドキュメントを開いてプリントを実行します。

  2. 出力先のプリンターを選択し、 [OK] をクリックします。
