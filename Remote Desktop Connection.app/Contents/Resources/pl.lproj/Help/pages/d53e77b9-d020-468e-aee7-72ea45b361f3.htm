Opcje uwierzytelniania

Pokaż wszystko

Funkcja uwierzytelniania komputera zdalnego sprawdza, czy połączenie jest nawiązywane z właściwym komputerem z systemem Windows. Dostępne są trzy opcje uwierzytelniania.

  • Zawsze łącz, nawet w przypadku niepowodzenia uwierzytelniania

    Usługa Podłączanie pulpitu zdalnego nawiązuje połączenie z komputerem z systemem Windows nawet wtedy, gdy nie może zweryfikować jego tożsamości. Ta opcja uwierzytelniania jest najmniej bezpieczna.

    Uwaga  Jeśli na komputerze z systemem Windows jest uruchomiony system Windows XP lub starsza wersja albo system Windows Server 2003 z dodatkiem Service Pack 1 (SP1) lub starsza wersja, komputer nie może przedstawić swojej tożsamości do zweryfikowania. Jeśli wiadomo, że na komputerze z systemem Windows jest uruchomiony jeden z tych wcześniejszych systemów operacyjnych, w celu uniknięcia wyświetlania ostrzeżeń związanych z uwierzytelnianiem można wybrać opcję Zawsze łącz, nawet w przypadku niepowodzenia uwierzytelniania.

  • Ostrzegaj w przypadku niepowodzenia uwierzytelniania (ustawienie domyślne)

    Jeśli usługa Podłączanie pulpitu zdalnego nie może zweryfikować tożsamości komputera z systemem Windows, wyświetla ostrzeżenie, aby użytkownik mógł zdecydować, czy nawiązywanie połączenia ma być kontynuowane. Ta opcja uwierzytelniania zapewnia większe bezpieczeństwo niż opcja Zawsze łącz, nawet w przypadku niepowodzenia uwierzytelniania.

  • Nie łącz w przypadku niepowodzenia uwierzytelniania

    Jeśli usługa Podłączanie pulpitu zdalnego nie może zweryfikować tożsamości komputera z systemem Windows, nawiązanie połączenia nie będzie możliwe. Ta opcja uwierzytelniania korzysta z rozszerzonych funkcji zabezpieczeń w przypadku nawiązywania połączenia z komputerem z uruchomionym systemem Windows Vista. Zapewnia ona również większe bezpieczeństwo niż dwie pozostałe opcje uwierzytelniania.

    Uwaga  Po wybraniu tej opcji nie będzie można nawiązać połączenia z komputerami, na których jest uruchomiony system operacyjny Windows w wersji wcześniejszej niż Windows Vista.
