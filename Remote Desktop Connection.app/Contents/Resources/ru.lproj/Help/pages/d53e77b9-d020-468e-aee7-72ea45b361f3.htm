Параметры проверки подлинности

Проверка подлинности удаленного компьютера позволяет убедиться в том, что устанавливается подключение к нужному компьютеру под управлением Windows. Доступны три способа проверки подлинности.

  • Подключаться, даже если проверка подлинности не прошла

    Подключение устанавливается даже в том случае, если подключение к удаленному рабочему столу не может проверить удостоверение компьютера под управлением Windows. Это наименее надежный вариант проверки подлинности.

    Примечание  Если на компьютере под управлением Windows запущена система Windows XP или более ранней версии либо Windows Server 2003 с пакетом обновления 1 (SP1) или более ранней версии, выполнить проверку удостоверения невозможно. Если известно, что на компьютере под управлением Windows установлена одна из этих более ранних версий операционной системы, можно избежать появления предупреждений о проверке подлинности, выбрав вариант Подключаться, даже если проверка подлинности не прошла.

  • Предупреждать, если проверка подлинности не прошла (вариант по умолчанию)

    Если подключение к удаленному рабочему столу не может проверить удостоверение компьютера под управлением Windows, пользователю выдается предупреждение и предлагается указать, следует ли устанавливать подключение. Этот вариант проверки подлинности более безопасен, чем вариант Подключаться, даже если проверка подлинности не прошла.

  • Не подключаться, если проверка подлинности не прошла

    Если подключение к удаленному рабочему столу не сможет проверить удостоверение компьютера под управлением Windows, установить подключение не удастся. Для этого способа проверки подлинности при подключении к компьютеру под управлением Windows Vista используются расширенные функции обеспечения безопасности. Этот вариант более безопасен, чем два других способа проверки подлинности.

    Примечание  Если выбран этот вариант, установить подключение к компьютерам под управлением ОС Windows более ранних версий, чем Windows Vista, не удастся.
