Ajout de vos nom d'utilisateur et mot de passe Windows au trousseau Mac OS

Tout afficher

Vous pouvez vous connecter à un ordinateur Windows plus rapidement en enregistrant vos informations de connexion dans le trousseau Mac OS. Les informations ajoutées au trousseau sont enregistrées dans fichier de connexionFichier contenant toutes les informations nécessaires pour une connexion à distance. Par exemple, vous pouvez définir la taille d'écran pour la fenêtre de session, ou partager des lecteurs Mac avec l'ordinateur Windows en enregistrant ces préférences dans un fichier de connexion. Si vous vous connectez à plusieurs ordinateurs Windows, vous pouvez enregistrer un fichier de connexion distinct pour chacun d'eux., de sorte que vous ne devez pas les entrer à chaque connexion.

Remarque  Les mots de passe ne sont enregistrés que dans le trousseau Mac OS, pas dans le fichier de connexion.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Cliquez sur Connexion.

  3. Tapez le nom d'utilisateur, le mot de passe et le domaine pour l'ordinateur Windows, puis activez la case à cocher Ajouter des informations utilisateur à votre trousseau.

    Conseil  Vous pouvez également enregistrer des informations utilisateur partielles dans votre trousseau. Par exemple, vous pouvez enregistrer le nom d'utilisateur et le domaine de façon à pouvoir ouvrir une session en ne tapant que le mot de passe.
