Windows のユーザー名とパスワードを Mac OS のキーチェーンに追加する

すべて表示

ログイン情報を Mac OS のキーチェーンに保存しておくと、Windows コンピューターにすばやく接続できます。キーチェーンに追加した情報は接続ファイルリモート接続に関するすべての情報を格納するファイル。たとえば、セッション ウインドウの画面サイズや Windows ベースのコンピューターから使用できるようにする Mac ドライブなどの設定は、接続ファイルに保存できます。Windows ベースの複数のコンピューターに接続する場合は、接続するコンピューターごとに接続ファイルを保存することができます。に保存されるので、接続のたびに情報を入力する必要はありません。

メモ  パスワードは Mac OS のキーチェーンのみに保存され、接続ファイルには保存されません。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [ [ログイン] ] をクリックします。

  3. Windows コンピューターのユーザー名、パスワード、およびドメインを入力してから、[ [Add user information to your keychain] ] チェック ボックスをオンにします。

    ヒント  一部のユーザー情報だけをキーチェーンに保存することもできます。たとえば、パスワードを入力するだけでログインできるように、ユーザー名とドメインを保存しておくことができます。
