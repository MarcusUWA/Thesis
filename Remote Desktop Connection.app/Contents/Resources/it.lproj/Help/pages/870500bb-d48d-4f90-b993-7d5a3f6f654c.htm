Novità di Connessione desktop remoto

Mostra tutto

Client di Connessione desktop remoto Microsoft per Mac 2 presenta un'interfaccia utente rinnovata, opzioni di personalizzazione migliorate e la possibilità di connettersi a più computer Windows contemporaneamente. Nella tabella seguente sono elencati tali miglioramenti.

Funzionalità

Descrizione

Universal binary

Viene eseguito su Mac Intel e PowerPC.

Interfaccia utente con barra dei menu

Semplifica la finestra di connessione ed espande la barra dei menu per includere i menu per le attività comuni (File, Modifica, Visualizza e ?). Le preferenze dell'applicazione sono presenti nel menu Connessione desktop remoto.

Remote Desktop Protocol 6.0

Migliora le prestazioni e la compatibilità con Windows Vista e Windows 7.

Sessioni multiple

È possibile connettersi a più computer quando si salva un file di connessione per ciascun computer. Utilizzare il nuovo menu File per aprire connessioni nuove, salvate e aperte in precedenza e gestire tutte le modifiche dei file di connessione.

Migliore personalizzazione di Connessione desktop remoto 1.0

Consente di accettare le modifiche apportate alle preferenze dell'applicazione, come i tasti di scelta rapida, durante una sessione di desktop remoto. Le modifiche vengono applicate alla successiva connessione.

Riconnessione automatica

Consente di ristabilire una sessione remota quando la connessione di rete viene persa.

Opzioni di ridimensionamento dello schermo

Consentono di eseguire la sessione di desktop remoto in una finestra ridimensionabile o in modalità schermo intero.

Stampa

Consente di stampare i documenti Windows su qualsiasi stampante connessa al computer Macintosh.

Autenticazione a livello di rete

Consente di verificare l'identità del computer Windows prima di stabilire una connessione di desktop remoto. È possibile selezionare questa opzione quando si esegue la connessione a un computer che esegue Windows Vista o Windows 7. L'autenticazione a livello di rete è più sicura delle opzioni di autenticazione nelle versioni precedenti di Windows.

Risoluzione schermo intero

Offre impostazioni per una risoluzione migliore a schermo intero.

Sessioni console server

La velocità di accesso viene aumentata quando si specifica "console" come parte del nome del computer.
