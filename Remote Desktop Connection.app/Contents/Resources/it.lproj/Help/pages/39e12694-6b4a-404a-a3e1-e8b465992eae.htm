Chiusura di una connessione

Mostra tutto

È possibile chiudere una sessione remota disconnettendosi dal computer Windows. La disconnessione comporta la chiusura della connessione, ma lascia aperti e in esecuzione i file o i programmi Windows in uso, affinché sia possibile tornarvi in un secondo momento. La chiusura della sessione termina la sessione. Qualsiasi applicazione in esecuzione durante la sessione viene chiusa mentre le modifiche non salvate nei file aperti verranno perse.

Chiusura di una connessione a un computer che esegue Windows 7

Per eseguire la disconnessione consentendo l'esecuzione delle applicazioni sul computer Windows, utilizzare Disconnetti. Per chiudere tutte le applicazioni ed eseguire la disconnessione da Windows, utilizzare Chiudi sessione.

Chiudi sessione

  • Fare clic su Start, quindi fare clic su Chiudi sessione.

Disconnetti

  • Fare clic su Start, fare clic sulla freccia accanto a Chiudi sessione, quindi fare clic cu Disconnetti.

Chiudi sessione

Chiusura di una connessione su un computer che esegue Windows XP

Per eseguire la disconnessione consentendo l'esecuzione delle applicazioni sul computer Windows, utilizzare Disconnetti. Per chiudere tutte le applicazioni ed eseguire la disconnessione da Windows, utilizzare Chiudi sessione.

Disconnetti

Chiudi sessione

  • Fare clic su Start, selezionare Chiudi sessione, quindi fare nuovamente clic su Chiudi sessione.
