Добавление имени пользователя и пароля Windows в цепочку ключей Mac OS

Показать все

Чтобы ускорить подключение к компьютеру под управлением Windows, можно сохранить учетные данные в цепочке ключей Mac OS. Сведения, добавляемые в цепочку ключей, сохраняются в файл подключенияФайл, содержащий все данные для удаленного подключения. Например, в файле подключения можно сохранить параметры размера экрана для окна сеанса или совместного использования дисков Mac с компьютерами под управлением системы Windows. При подключении к нескольким компьютерам с системой Windows можно сохранить отдельный файл подключения для каждого из них., поэтому их не нужно вводить при каждом подключении.

Примечание  Пароли сохраняются только в цепочке ключей Mac OS, но не в файле подключения.

  1. В меню RDC выберите пункт Настройки.

  2. Выберите команду Вход.

  3. Введите имя пользователя, пароль и домен для компьютера под управлением Windows и установите флажок Добавить данные пользователя в цепочку ключей.

    Совет  Кроме того, можно сохранить в цепочке ключей часть данных пользователя. Например, можно сохранить имя пользователя и домен, чтобы при подключении вводить только пароль.
