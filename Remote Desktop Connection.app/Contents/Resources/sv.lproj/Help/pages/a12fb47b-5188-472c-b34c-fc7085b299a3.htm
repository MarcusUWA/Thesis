Ansluta till en Windows-baserad dator

Visa alla

Innan du kan ansluta måste du aktivera fjärrskivbordsinställningen på den dator som du vill ansluta till. Här följer en beskrivning som visar hur du gör för att ansluta till en Windows-baserad dator från din Mac.

Obs!  Det går inte att använda ett smartkort med Microsoft Fjärrskrivbordanslutningsklient för Mac 2.

Steg 1: Tillåt fjärranslutningar på den Windows-baserade datorn

Välj en av följande procedurer beroende på vilken version av Windows som körs på den Windows-baserade dator som du vill ansluta till.

Tillåta fjärranslutningar i Windows 7

Viktigt!

Du måste vara administratör på den Windows-baserade datorn för att kunna ändra inställningarna för fjärrskrivbord.

  1. Klicka på Start  och sedan på Kontrollpanelen på en dator som kör Windows.

  2. Klicka på System och säkerhet och klicka sedan på Tillåt fjärråtkomst under System.

  3. Klicka på önskat alternativ under Fjärrskrivbord.

    Skriv lösenordet eller bekräfta om du blir ombedd om ett administratörslösenord eller en bekräftelse.

    Obs!  Autentisering på nätverksnivå förbättrar säkerheten eftersom funktionen kräver att du anger din inloggningsinformation för Windows på Mac innan fjärranslutningen initieras. Mer information om autentisering på nätverksnivå finns i hjälpen för Windows.

  4. Klicka på Markera användare.

  5. Gör något av följande:

    Om du vill

    Gör du så här

    Verifiera att du har åtkomst

    Se till att ditt användarkonto står under rutan. Om du är administratör för datorn finns du med automatiskt.

    Lägga till ett annat användarkonto

    Klicka på Lägg till och ange sedan det användarkonto som du vill använda.

Steg 2: Hitta den Windows-baserade datorns namn

När du öppnar Anslutning till fjärrskrivbord måste du ange namnet på den Windows-baserade dator som du vill ansluta till. Välj en av följande procedurer beroende på vilken version av Windows som körs på den Windows-baserade dator som du vill ansluta till.

Hitta datornamnet i Windows 7

  1. Klicka på Start  och sedan på Kontrollpanelen på en dator som kör Windows.

  2. Klicka på System och säkerhet och markera sedan Visa namnet på datorn under System.

  3. Under Inställningar för datornamn, domän och arbetsgrupp noterar du datornamnet eller det fullständiga datornamnet om den Windows-baserade datorn är medlem i en domän.

    När du öppnar Anslutning till fjärrskrivbord på Mac måste du ange det här namnet.

Hitta datornamnet i Windows Vista

  1. Klicka på Start  och sedan på Kontrollpanelen på en dator som kör Windows.

    Klicka på Kontrollpanelen – startsida i navigeringsfönstret om du använder klassiskt läge.

  2. Klicka på System och underhåll och klicka sedan på System.

  3. Under Inställningar för datornamn, domän och arbetsgrupp noterar du datornamnet eller det fullständiga datornamnet om den Windows-baserade datorn är medlem i en domän.

    När du öppnar Anslutning till fjärrskrivbord på Mac måste du ange det här namnet.

Hitta datornamnet i Windows XP

  1. Klicka på Start, högerklicka på Den här datorn och klicka sedan på Egenskaper på den Windows-baserade datorn.

  2. Notera det fullständiga datornamnet på fliken Datornamn.

    När du öppnar Anslutning till fjärrskrivbord på Mac måste du ange det här namnet.

Steg 3: Öppna en ny anslutning

Du kan öppna en anslutning till en enda Windows-baserad dator eller till flera Windows-baserade datorer samtidigt.

Viktigt!

Den Windows-baserade datorn måste vara på, och det måste finnas en nätverksanslutning. Du kan inte ansluta till en dator som är i viloläge eller vänteläge. Se till att inställningarna för viloläge och vänteläge på den Windows-baserade datorn är inställda på Aldrig. Om du vill veta hur du gör de här ändringarna läser du i hjälpen för Windows. Kontrollera att den dator som du vill ansluta till kör en version av Windows som stödjer fjärranslutningar. En fullständig lista finns på Kompatibilitet med Windows-versioner.

  1. Öppna Anslutning till fjärrskrivbord på din Macintosh-dator.

  2. Skriv namnet på den Windows-baserade dator som du vill ansluta till i rutan Dator och klicka sedan på Anslut.

  3. Ange användarnamn, lösenord och domän för den Windows-baserade datorn om de krävs.
