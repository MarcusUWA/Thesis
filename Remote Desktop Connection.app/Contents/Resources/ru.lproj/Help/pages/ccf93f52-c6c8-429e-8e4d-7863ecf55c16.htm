Возможности универсального доступа

Показать все

В подключение к удаленному рабочему столу доступны некоторые возможности универсального доступа Mac OS X. Возможности универсального доступа облегчают использование компьютера людям с ограничениями зрения, слуха и подвижности.

Чтобы настроить параметры доступа, в меню Apple выберите пункт Системные настройки, а затем в разделе Система щелкните элемент Универсальный доступ. В таблице ниже перечислены возможности универсального доступа Mac OS X, которые поддерживает подключение к удаленному рабочему столу.

Функции Mac OS X недоступны на рабочем столе Windows после запуска сеанса удаленного рабочего стола. Сведения о специальных возможностях, доступных в Windows, см. на веб-странице Специальные возможности (Майкрософт) по адресу http://www.microsoft.com/enable (на английском языке).

VoiceOver

Звуковое воспроизведение большинства команд меню, диалоговых окон и других элементов подключение к удаленному рабочему столу. Эта функция не работает для содержимого окна сеанса удаленного рабочего стола. Например, после подключения к компьютеру под управлением Windows функция VoiceOver не будет воспроизводить элементы на рабочем столе Windows в виде звуков.

Зрения

Масштаб

Эта функция позволяет изменять масштаб изображения на экране, увеличивая его.

Зрения

Отображение

Эта функция позволяет изменить режим отображения содержимого на экране (черный текст на белом фоне или белый текст на черном фоне) и другие настройки контрастности.

Мышь

Полный доступ с клавиатуры

Эта функция позволяет использовать для перемещения между элементами управления клавиатуру вместо мыши.

Примечание  Описания вкладок соответствуют принятой в США раскладке клавиатуры. В других раскладках могут использоваться другие клавиши. Кроме того, могут быть другими сочетания клавиш для ноутбуков.

Дополнительные сочетания клавиш см. в справке Mac.
