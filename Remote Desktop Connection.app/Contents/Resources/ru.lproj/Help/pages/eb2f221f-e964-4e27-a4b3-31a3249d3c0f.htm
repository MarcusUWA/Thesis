Печать документа

Показать все

Документы, которые хранятся на компьютере под управлением Windows, можно печатать на принтере, подключенном к компьютеру Mac. Для этого нужно сначала разрешить доступ к этому принтеру с компьютера под управлением Windows.

После того как доступ к принтеру будет разрешен, к нему можно будет обращаться с компьютера под управлением Windows. Если сохранить файл подключенияФайл, содержащий все данные для удаленного подключения. Например, в файле подключения можно сохранить параметры размера экрана для окна сеанса или совместного использования дисков Mac с компьютерами под управлением системы Windows. При подключении к нескольким компьютерам с системой Windows можно сохранить отдельный файл подключения для каждого из них., а затем снова открыть удаленный сеанс с его помощью, принтер останется доступным.

Выполните одно из перечисленных ниже действий.

Разрешение доступа к принтеру с компьютера под управлением Windows

  1. В меню RDC выберите пункт Настройки.

  2. Выберите команду Принтеры и установите флажок Использовать принтер, подключенный к компьютеру Mac.

  3. Выберите нужный принтер.

    Советы 
     

    • Чтобы задать принтер во время печати, выберите пункт Все принтеры.

    • Чтобы сохранить файл подключения, в котором разрешен доступ к принтеру, в меню Файл выберите команду Сохранить как и введите имя файла подключения.

Печать документа с компьютера под управлением Windows на принтере, подключенном к компьютеру Mac

  1. На рабочем столе Windows откройте нужный документ и отправьте его на печать.

  2. Выберите принтер для печати документа и нажмите кнопку ОК.
