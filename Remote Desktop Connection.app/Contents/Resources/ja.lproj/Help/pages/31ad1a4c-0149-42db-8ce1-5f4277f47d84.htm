コンピューター間でファイルをコピーする

すべて表示

ファイルをコピーするには、まず Windows コンピューター上で Mac のディスクまたはフォルダーを開いてから、必要なファイルを Windows コンピューターにコピーする必要があります。Mac から Windows コンピューターにファイルをドラッグすることはできません。

手順 1: Mac のディスクまたはフォルダーを Windows コンピューターに公開する

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [ドライブ] をクリックし、ポップアップ メニューで必要なオプションを選択します。

ディスクまたはフォルダーが利用可能になり、Windows ベースのコンピューターから Mac 上のファイルにアクセスできるようになります。

手順 2: コンピューター間でファイルをコピーする

作業対象の Mac のディスクまたはフォルダーが Windows コンピューターに公開された後に、コンピューター間でファイルをコピーできます。

  1. [スタート] メニューの [コンピューター] または [マイ コンピューター] をクリックします。

    利用可能になった Mac のディスクまたはフォルダーが、Windows ベースのコンピューター上の他のすべてのディスクやフォルダーと共に一覧に表示されます。

  2. コピー元のディスクまたはフォルダーを開きます。

  3. [スタート] メニューの [コンピューター] または [マイ コンピューター] をクリックします。

  4. コピー先のディスクまたはフォルダーを開きます。

    これで、2 つのウインドウが開いて、コピー元の場所とコピー先の場所が表示されている状態になります。

  5. コピー元のドキュメントをコピー先の場所までドラッグします。
