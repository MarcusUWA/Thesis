Windows コンピューターへの接続

すべて表示

接続する前に、まず接続先のコンピューターでリモート デスクトップ設定を有効にする必要があります。Mac から Windows コンピューターに接続するには、以下に示す手順を実行します。

メモ  Microsoft Remote Desktop Connection Client for Mac 2 では、スマート カードは使用できません。

手順 1: Windows コンピューターでリモート接続を許可する

接続先の Windows ベースのコンピューターで実行されている Windows のバージョンに基づいて、次のいずれかの手順を実行します。

Windows 7 でリモート接続を許可する

重要

リモート デスクトップの設定を変更するには、Windows ベースのコンピューターの管理者権限が必要です。

  1. Windows ベースのコンピューターで、 [スタート] ボタン () をクリックし、 [コントロール パネル] をクリックします。

  2. [ [システムとセキュリティ] ] をクリックし、[ [システム] ] で [ [リモート アクセスの許可] ] をクリックします。

  3. [リモート デスクトップ] で、必要なオプションをクリックします。

    管理者のパスワードを求めるメッセージが表示されたらパスワードを入力し、その後の確認メッセージに従って、もう一度パスワードを入力します。

    メモ  ネットワーク レベル認証によって、Mac からリモート接続を開始する際に Windows のログイン情報の入力が必要になるため、セキュリティが向上します。ネットワーク レベル認証の詳細については、Windows ヘルプを参照してください。

  4. [ユーザーの選択] をクリックします。

  5. 必要に応じて、次の操作を行います。

    目的

    操作

    アクセス許可を確認する

    自分のユーザー アカウントがボックスの一覧に表示されていることを確認します。コンピューターの管理者のアカウントは自動的に一覧に表示されます。

    他のユーザー アカウントを追加する

    [追加] をクリックし、ボックス内に、追加するユーザー アカウント名を入力します。

  4. [ユーザーの選択] をクリックします。

  5. 必要に応じて、次の操作を行います。

    操作

Windows XP でリモート接続を許可する

重要

リモート デスクトップの設定を変更するには、Windows ベースのコンピューターの管理者権限が必要です。

  1. Windows ベースのコンピューターで、 [スタート] ボタンをクリックし、 [コントロール パネル] をクリックします。

    クラシック表示の場合は、ナビゲーション ウインドウの [カテゴリの表示に切り替える] をクリックします。

  2. [パフォーマンスとメンテナンス] をクリックし、 [システム] をクリックします。

  3. [リモート] タブで、 [このコンピューターにユーザーがリモートで接続することを許可する] チェック ボックスをオンにします。

  4. [リモート ユーザーの選択] をクリックします。

  5. 必要に応じて、次の操作を行います。

    目的

    操作

    アクセス許可を確認する

    自分のユーザー アカウントがボックスの一覧に表示されていることを確認します。コンピューターの管理者のアカウントは自動的に一覧に表示されます。

    他のユーザー アカウントを追加する

    [追加] をクリックし、ボックス内に、追加するユーザー アカウント名を入力します。

手順 2: Windows コンピューターの名前を調べる

リモート デスクトップ接続 を開いたときに、接続先の Windows コンピューターの名前を入力する必要があります。接続先の Windows ベースのコンピューターで実行されている Windows のバージョンに基づいて、次のいずれかの手順を実行します。

Windows 7 でコンピューター名を調べる

  1. Windows ベースのコンピューターで、 [スタート] ボタン () をクリックし、 [コントロール パネル] をクリックします。

  2. [ [システムとセキュリティ] ] をクリックし、[ [システム] ] で [ [コンピューターの名前の参照] ] をクリックします。

  3. [コンピューター名、ドメインおよびワークグループの設定] に表示されているコンピューター名を控えておきます。Windows ベースのコンピューターがドメインのメンバーの場合はフル コンピューター名を控えておきます。

    この名前は、Mac で リモート デスクトップ接続 を起動したときに入力します。

Windows Vista でコンピューター名を調べる

  1. Windows ベースのコンピューターで、 [スタート] ボタン () をクリックし、 [コントロール パネル] をクリックします。

    クラシック表示の場合は、ナビゲーション ウインドウの [コントロール パネル ホーム] をクリックします。

  2. [システムとメンテナンス] をクリックし、 [システム] をクリックします。

  3. [コンピューター名、ドメインおよびワークグループの設定] に表示されているコンピューター名を控えておきます。Windows ベースのコンピューターがドメインのメンバーの場合はフル コンピューター名を控えておきます。

    この名前は、Mac で リモート デスクトップ接続 を起動したときに入力します。

Windows XP でコンピューター名を調べる

  1. Windows コンピューターで、[ [スタート] ] をクリックし、[ [マイ コンピュータ] ] を右クリックして [ [プロパティ] ] をクリックします。

  2. [コンピューター名] タブに表示されているフル コンピューター名を控えておきます。

    この名前は、Mac で リモート デスクトップ接続 を起動したときに入力します。

手順 3: 新しい接続を開く

1 台の Windows コンピューターへの接続を開くか、または複数の Windows コンピューターへの接続を同時に開くことができます。

重要

Windows ベースのコンピューターは、電源がオンで、ネットワーク接続が確立された状態である必要があります。スリープ状態、休止状態、スタンバイ状態のコンピューターに接続することはできません。したがって、Windows ベースのコンピューターのスリープ、休止、およびスタンバイの設定は必ず [なし] に設定してください。スリープ、休止、およびスタンバイの設定の変更方法については、Windows のヘルプを参照してください。接続先のコンピューターで実行している Windows が、リモート接続をサポートするバージョンであることも確認してください。バージョンの一覧については、「Windows のバージョン互換性」を参照してください。

  1. Macintosh コンピューターで、リモート デスクトップ接続 を起動します。

  2. [コンピューター] ボックスに接続先の Windows ベースのコンピューターの名前を入力し、 [接続] をクリックします。

  3. 必要に応じて、Windows ベースのコンピューターのユーザー名、パスワード、およびドメインを入力します。
