Не удается подключиться к компьютеру под управлением Windows

Показать все

Причина На компьютере под управлением Windows не разрешены удаленные подключения.

Решение Разрешите доступ к удаленному рабочему столу на компьютере, к которому требуется подключиться.

Включение параметра удаленного рабочего стола в ОС Windows 7

Важно

Изменять параметры удаленного рабочего стола может только администратор компьютера под управлением Windows.

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

  2. Щелкните элемент Система и безопасность, а затем в разделе Система выберите команду Настройка удаленного доступа.

  3. В разделе Удаленный рабочий стол выберите нужный параметр.

    При запросе пароля администратора или его подтверждения введите пароль или подтвердите его.

    Примечание  Проверка подлинности на уровне сети повышает уровень безопасности, поскольку перед открытием удаленного подключения на компьютере Mac требуется ввести регистрационные данные Windows. Дополнительные сведения о проверке подлинности на уровне сети см. в справке Windows.

  4. Нажмите кнопку Выбрать пользователей.

  5. Выполните любое из указанных ниже действий.

    Чтобы

    Выполните следующие действия

    Проверить доступ

    Проверьте, указана ли ваша учетная запись пользователя в списке под полем. Учетная запись администратора компьютера помещается в список автоматически.

    Добавить другую учетную запись пользователя

    Нажмите кнопку Добавить и введите нужное имя учетной записи в поле.

Создание исключений в брандмауэре Windows на компьютере под управлением Windows 7

  3. В списке Разрешенные программы и компоненты установите флажок Удаленный рабочий стол.

Создание исключений в брандмауэре Windows на компьютере с Windows XP

Важно

Изменять параметры брандмауэра может только администратор компьютера под управлением Windows.

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

    В классическом виде в области навигации выберите пункт Переключение к виду по категориям.

  2. Щелкните элемент Центр обеспечения безопасности и в разделе Настройка параметров безопасности выберите команду Брандмауэр Windows.

  3. Снимите флажок Не разрешать исключения.

  4. Откройте вкладку Исключения и установите флажок Удаленный рабочий стол.

Причина Компьютеру Mac не удается подключиться к сети.

Решение Убедитесь в том, что компьютер под управлением Windows включен и подключен к сети.

Сбой удаленного подключения может объясняться наличием проблем в сети. Например, такой сбой происходит в ситуации, когда удаленная сеть не работает или на компьютере под управлением Windows отключен кабель Ethernet.

Причина Политика сети блокирует подключения.

Решение Уточните у администратора сети, какая политика безопасности используется в организации.

В организациях используются различные политики безопасности. Например, чтобы подключиться к компьютеру под управлением Windows в сети, в которой для обеспечения безопасности сетевого трафика используется протокол IPsec, для этого компьютера необходимо создать граничное исключение.

Причина Не удается найти компьютер под управлением Windows.

Решение Проверьте, правильно ли настроены на компьютере Mac параметры сети для используемого типа подключения.

На компьютере Mac должны быть настроены параметры подключения Ethernet или беспроводного подключения для сети, с которой требуется установить соединение.

Решение Проверьте, правильно ли введено имя компьютера.

Проверка имени компьютера в ОС Windows 7

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

  2. Щелкните элемент Система и безопасность, а затем в разделе Система выберите команду Просмотр имени этого компьютера.

  3. В разделе Имя компьютера, имя домена и параметры рабочей группы уточните имя компьютера или полное имя компьютера, если компьютер под управлением Windows является членом домена.

    При запуске подключение к удаленному рабочему столу на компьютере Mac необходимо ввести это имя.

Проверка имени компьютера в ОС Windows Vista

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

    В классическом виде в области навигации выберите пункт Панель управления — домашняя страница.

  2. Щелкните элемент Система и ее обслуживание, а затем — Система.

  3. В разделе Имя компьютера, имя домена и параметры рабочей группы уточните имя компьютера или полное имя компьютера, если компьютер под управлением Windows является членом домена.

    При запуске подключение к удаленному рабочему столу на компьютере Mac необходимо ввести это имя.

Проверка имени компьютера в ОС Windows XP

  1. На компьютере под управлением Windows нажмите кнопку Пуск, щелкните правой кнопкой мыши элемент Мой компьютер и выберите пункт Свойства.

  2. На вкладке Имя компьютера уточните полное имя компьютера.

    При запуске подключение к удаленному рабочему столу на компьютере Mac необходимо ввести это имя.

Решение Если не удается установить подключение по имени компьютера, используйте его IP-адрес.

  1. На компьютере под управлением Windows нажмите кнопку Пуск и последовательно выберите пункты Все программы, Стандартные и Командная строка.

  2. В командной строке введите ipconfig и нажмите клавишу ВВОД.

  3. В разделе Подключение по локальной сети через адаптер Ethernet найдите IP-адрес компьютера.

    Открыв подключение к удаленному рабочему столу на компьютере Mac, можно вместо имени компьютера указать его IP-адрес. На компьютерах под управлением Windows Vista или Windows 7 IP-адрес указан в формате IPv4.

Решение Убедитесь в том, что компьютер Mac подключен к Интернету.

  • Откройте веб-браузер (например, Safari) и убедитесь в том, что в нем удается просматривать веб-страницы.

Причина Не удается проверить удостоверение компьютера под управлением Windows.

Решение Убедитесь в том, что параметры безопасности подключение к удаленному рабочему столу совместимы с версией Windows, установленной на компьютере, к которому выполняется подключение.

Если на компьютере под управлением Windows запущена система Windows XP или более ранней версии либо Windows Server 2003 с пакетом обновления 1 (SP1) или более ранней версии, выполнить проверку удостоверения невозможно. Если на удаленном компьютере установлена одна из этих версий Windows, а параметры безопасности на компьютере Mac требуют перед подключением выполнять проверку подлинности, удаленное подключение не будет установлено.

  1. В меню RDC выберите пункт Настройки, а затем — Безопасность.

  2. Убедитесь в том, что флажок Не подключаться, если проверка подлинности не прошла снят.
