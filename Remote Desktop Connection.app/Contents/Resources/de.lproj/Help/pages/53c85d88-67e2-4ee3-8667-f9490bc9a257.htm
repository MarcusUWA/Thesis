Kompatibilität mit Windows-Versionen

Alles anzeigen

Mit der Remotedesktopverbindung können Sie eine Verbindung mit Computern herstellen, auf denen folgende Versionen von Windows ausgeführt werden:

Windows 7

  • Windows 7 Professional

  • Windows 7 Enterprise

  • Windows 7 Ultimate

Windows Vista

  • Windows Vista Business

  • Windows Vista Enterprise

  • Windows Vista Ultimate

Windows XP-Editionen

  • Windows XP Professional

  • Windows XP Media Center

Windows Server 2008-Editionen

  • Windows Server 2008 R2

  • Windows Server 2008 Datacenter

  • Windows Server 2008 Enterprise

  • Windows Server 2008 Standard

Windows Server 2003-Editionen

  • Windows Server 2003 Datacenter

  • Windows Server 2003 Enterprise

  • Windows Server 2003 Standard
