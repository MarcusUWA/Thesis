Impossibilité d'afficher la barre de menus Mac

Tout afficher

Cause : La session Bureau à Distance s'exécute en mode plein écran.

Solution : Pointez sur le bord de l'écran où la barre de menus se trouve habituellement.

La barre de menus est masquée en mode plein écran afin de maximiser l'espace écran disponible.

Cause : Les préférences d'affichage pour le mode plein écran sont définies de façon à ne pas afficher la barre de menus et le Dock.

Solution : Désactivez l'option plein écran dans les préférences d'affichage.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Cliquez sur Affichage, puis désactivez la case à cocher Ne pas afficher le Dock et la barre de menus Mac en mode plein écran.
