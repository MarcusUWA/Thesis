Drucken eines Dokuments

Alles anzeigen

Dokumente auf dem Windows-basierten Computer können auf einem Drucker ausgegeben werden, der mit dem Macintosh-Computer verbunden ist.

Nachdem Sie den Drucker verfügbar gemacht haben, können Sie vom Windows-basierten Computer aus darauf zugreifen. Wenn Sie die VerbindungsdateiEine Verbindungsdatei enthält alle Informationen für eine Remoteverbindung. Beispielsweise können Sie die Größe des Sitzungsfensters festlegen, oder Sie können Macintosh-Laufwerke für Windows-basierte Computer freigeben, indem Sie diese Einstellungen in einer Verbindungsdatei speichern. Wenn Sie Verbindungen zu mehreren Windows-basierten Computern herstellen, können Sie für jeden Computer eine eigene Verbindungsdatei speichern. speichern und diese für die Anmeldung in nachfolgenden Remotesitzungen verwenden, bleibt der Drucker weiterhin verfügbar.

Führen Sie eine der folgenden Aktionen aus:

Verfügbarmachen eines Druckers für den Windows-basierten Computer

  1. Klicken Sie im Menü RDC auf Einstellungen.

  2. Klicken Sie auf Drucker, und aktivieren Sie das Kontrollkästchen Drucker verwenden, der mit dem Macintosh verbunden ist.

  3. Wählen Sie den gewünschten Drucker aus.

    Tipps 
     

    • Wählen Sie Alle Drucker aus, wenn Sie den Drucker zum Zeitpunkt des Drucks bestimmen möchten.

    • Wenn Sie eine Verbindungsdatei mit Zugriff auf den Drucker speichern möchten, klicken Sie im Menü Datei auf Speichern unter, und geben Sie einen Namen für die Verbindungsdatei ein.

Drucken eines Windows-basierten Dokuments auf einem Drucker mit Verbindung zu einem Macintosh

  1. Öffnen Sie das zu druckende Dokument auf dem Windows-Desktop, und drucken Sie dieses.

  2. Wählen Sie dazu den gewünschten Drucker aus, und klicken Sie auf OK.
