接続を終了する

すべて表示

Windows コンピューターから切断またはログオフすることによってリモート セッションを閉じることができます。切断すると接続は閉じますが、Windows で開いていたファイルや実行中のプログラムはそのままの状態であり、再接続すれば作業を再開できます。ログオフするとセッションが終了します。そのセッションで実行中であったすべてのアプリケーションは閉じられ、開いていたファイルの未保存の変更は失われます。

Windows 7 が動作しているコンピューターへの接続を閉じる

Windows ベースのコンピューター上でプログラムを実行したまま接続を切断するには、[切断] を使用します。すべてのプログラムを終了して Windows からログオフするには、[ログオフ] を使用します。

ログオフ

  • [ [スタート] ] をクリックし、[ [ログオフ] ] をクリックします。

切断

  • [ [スタート] ] をクリックし、[ [ログオフ] ] の横にある三角形をクリックして [ [切断] ] をクリックします。

Windows Vista が動作しているコンピューター上で接続を閉じる

Windows ベースのコンピューター上でプログラムを実行したまま接続を切断するには、[切断] を使用します。すべてのプログラムを終了して Windows からログオフするには、[ログオフ] を使用します。

切断

  • [ [スタート] ] をクリックし、[ [切断] ] をクリックします。

ログオフ

  • [ [スタート] ] をクリックし、[ロック] の横にある三角形をクリックして [ [ログオフ] ] をクリックします。

Windows XP が動作しているコンピューター上で接続を閉じる

Windows ベースのコンピューター上でプログラムを実行したまま接続を切断するには、[切断] を使用します。すべてのプログラムを終了して Windows からログオフするには、[ログオフ] を使用します。

切断

  • [ [スタート] ] をクリックし、[ [切断] ] をクリックしてから再び [ [切断] ] をクリックします。

ログオフ

  • [スタート] ボタンをクリックして [ログオフ] をクリックし、もう一度 [ログオフ] をクリックします。

Windows Server 2003 が動作しているコンピューター上で接続を閉じる

Windows ベースのコンピューター上でプログラムを実行したまま接続を切断するには、[切断] を使用します。すべてのプログラムを終了して Windows からログオフするには、[ログオフ] を使用します。

切断

  • Remote Desktop セッションのウィンドウで、 をクリックしてから [ [切断] ] をクリックします。

ログオフ

  • [スタート] ボタンをクリックして [ログオフ] をクリックし、もう一度 [ログオフ] をクリックします。
