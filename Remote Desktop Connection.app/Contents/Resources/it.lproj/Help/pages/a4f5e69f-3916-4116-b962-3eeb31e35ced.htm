Preferenze di visualizzazione di Connessione desktop remoto

Mostra tutto

È possibile salvare le preferenze di visualizzazione per ciascun computer Windows a cui si esegue la connessione. La maggior parte di queste preferenze influisce sulla visualizzazione dei grafici. Le opzioni selezionate possono influire notevolmente sulle prestazioni della sessione di desktop remotoConnessione tra un computer client (Mac) e un computer Windows remoto. Questa connessione può avvenire nella stessa rete o in Internet. Quando viene creata una sessione, il desktop remoto (del computer Windows) viene visualizzato sotto forma di finestra sul desktop Mac. La sessione termina quando ci si disconnette dal computer Windows.. Le opzioni predefinite offrono le prestazioni migliori su una connessione a banda larga.

Impostazione predefinita

Descrizione

Dimensioni desktop remoto

1024x768

Consente di modificare la risoluzione dello schermo e le dimensioni della finestra in cui è visualizzata la sessione di desktop remoto. Ad esempio, per visualizzare la sessione in una finestra più piccola rispetto alle dimensioni predefinite, selezionare un valore numerico più piccolo per questa opzione.

Suggerimento  Per modificare le dimensioni della finestra durante una sessione, scegliere Adatta alla finestra dal menu Visualizza, quindi trascinare il controllo di ridimensionamento fino a raggiungere la dimensione desiderata per la finestra.

Colori

Migliaia

Consente di regolare l'intensità del colore per la sessione. A seconda dei criteri della rete e della versione di Windows in esecuzione sul computer remoto, l'intensità massima del colore per la connessione remota potrebbe essere limitata. Se l'intensità del colore selezionata è maggiore rispetto all'impostazione consentita, Connessione desktop remoto crea la connessione all'intensità di colore massima consentita.

Apri finestra desktop remoto in

Monitor principale

Consente di aprire Connessione desktop remoto sul monitor specificato. Se il Mac è impostato per utilizzare due o più monitor, è possibile selezionare il monitor su cui si desidera aprire la finestra della sessione.

Mostra sfondo del desktop

Disattivato

Consente di visualizzare lo sfondo dal desktop Windows nella finestra della sessione.

Mostra caratteri smussati

Attivato

Consente di migliorare la leggibilità dei caratteri visualizzati smussando le diagonali frastagliate e le curve nelle lettere.

Visualizza contenuto della finestra durante il trascinamento

Disattivato

Consente di visualizzare il contenuto di una finestra che viene spostata nella schermata, anziché visualizzare solo la struttura della finestra.

Mostra menu e animazione della finestra

Disattivato

Consente di visualizzare menu a cascata e scorrere gli effetti di transizione per i menu e le descrizioni di comandi.

Mostra temi

Attivato

Consente di visualizzare le icone, l'audio, lo sfondo e altri elementi del computer Windows nella finestra di sessione.

Consenti acquisizione bitmap

Attivato

Consente di migliorare le prestazioni riducendo le informazioni inviate mediante la connessione.

Non mostrare la barra dei menu e il Dock Mac in modalità schermo intero

Disattivato

Consente di evitare che la barra dei menu e il Dock vengano visualizzati quando la finestra della sessione è in modalità schermo intero. Ad esempio, è possibile selezionare questa opzione per evitare che il Dock oscuri la barra delle applicazioni di Windows.
