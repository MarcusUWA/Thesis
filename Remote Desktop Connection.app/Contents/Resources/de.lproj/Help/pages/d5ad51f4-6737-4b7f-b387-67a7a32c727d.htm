Akustische Signale können nicht wiedergegeben werden

Alles anzeigen

Ursache: Die Audioeinstellungen sind deaktiviert.

Lösung: Wählen Sie eine Option aus, um akustische Signale wiederzugeben.

  1. Klicken Sie im Menü RDC auf Einstellungen.

  2. Klicken Sie auf Akustische Signale und anschließend auf die gewünschte Option.

Ursache: Die Gruppenrichtlinie für Terminaldienste lässt keine Weiterleitung von akustischen Signalen von dem Windows-basierten Computer zu.

Lösung: Informationen zur Gruppenrichtlinie für Terminaldienste erhalten Sie vom Netzwerkadministrator.

Alle Einstellungen für Remotedesktopverbindung werden durch die Gruppenrichtlinie überschrieben. Wenn die Gruppenrichtlinie für Terminaldienste keine Weiterleitung von akustischen Signalen zulässt, können akustische Signale weder auf dem Windows-basierten Computer noch auf dem Macintosh wiedergegeben werden.

Ursache: Die Lautsprecherlautstärke ist niedrig oder wurde stummgeschaltet.

Lösung: Überprüfen Sie die Soundeinstellungen für den Macintosh sowie für den Windows-basierten Computer.

Wenn die Lautstärke auf einem der beiden Computer deaktiviert wurde, können keine akustischen Signale wiedergegeben werden.
