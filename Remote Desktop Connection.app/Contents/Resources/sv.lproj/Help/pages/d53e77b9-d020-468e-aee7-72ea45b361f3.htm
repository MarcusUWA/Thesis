Autentiseringsalternativ

Visa alla

Med fjärrdatorautentisering verifierar du att du ansluter till rätt Windows-baserad dator. Det finns tre autentiseringsalternativ tillgängliga.

  • Anslut alltid, även om autentiseringen misslyckas

    Även om Anslutning till fjärrskrivbord inte kan verifiera den Windows-baserade datorns identitet ansluter den ändå. Det här är det osäkraste sättet att autentisera.

    Obs!  Om en Windows-baserad dator kör Windows XP eller tidigare, eller Windows Server 2003 Service Pack 1 (SP1) eller tidigare kan den inte tillhandahålla identiteten för verifikation.Om du vet att den Windows-baserade datorn körs med ett tidigare operativsystem kan du undvika att få autentiseringsvarningar genom att välja Anslut alltid, även om autentiseringen misslyckas.

  • Varna mig om autentiseringen misslyckas (standardinställning)

    Om Anslutning till fjärrskrivbord inte kan verifiera den Windows-baserade datorns identitet visas en varning så att du kan välja om anslutningen ska upprättas. Det här autentiseringsalternativet ger en bättre säkerhet än alternativet Anslut alltid, även om autentiseringen misslyckas.

  • Anslut inte om autentiseringen misslyckas

    Om Anslutning till fjärrskrivbord inte kan verifiera den Windows-baserade datorns identitet så kan du inte ansluta. Det här autentiseringsalternativet använder förbättrade säkerhetsfunktioner när du ansluter till en dator som kör Windows Vista. Det här autentiseringsalternativet förbättrar säkerheten jämfört med de andra två autentiseringsalternativen.

    Obs!  Om du markerar det här alternativet kan du inte ansluta till datorer som kör versioner av operativsystemet Windows som är tidigare än Windows Vista.
