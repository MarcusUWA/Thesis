Impossibilité d'entendre les signaux audio

Tout afficher

Cause : Les préférences des signaux audio sont désactivées.

Solution : Sélectionnez une option pour la lecture du signal audio.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Cliquez sur Son, puis sélectionnez l'option souhaitée.

Cause : La stratégie de groupe de services Terminal ne permet pas une redirection du signal audio à partir de l'ordinateur Windows.

Solution : Contactez l'administrateur réseau concernant votre stratégie de groupe de services Terminal.

La stratégie de groupe remplace toute préférence définie dans Connexion Bureau à Distance. Si la stratégie de groupe de services Terminal interdit la redirection du signal audio, celui-ci n'est lu ni sur l'ordinateur Windows ni sur l'ordinateur Mac.

Cause : Le volume du haut-parleur est bas ou « muet ».

Solution : Contrôlez les paramètres audio des ordinateurs Mac et Windows.

Si le volume est au minimum sur chaque ordinateur, le signal sonore n'est pas lu.
