Закрытие подключения

Показать все

Чтобы закрыть удаленный сеанс, можно отключиться от компьютера под управлением Windows или выйти из системы на нем. При отключении соединение прерывается, однако все используемые программы и файлы Windows остаются открытыми и к ним можно вернуться позднее. При выходе из системы сеанс завершается, все запущенные в нем приложения закрываются, а все несохраненные изменения в открытых файлах теряются.

Закрытие подключения к компьютеру под управлением Windows 7

Чтобы закрыть подключение, но при этом оставить работающие приложения на компьютере под управлением Windows, выберите команду "Отключение". Чтобы закрыть все приложения и выйти из Windows, выберите команду "Выход из системы".

Выход

  • Нажмите кнопку Пуск и выберите команду Выход из системы.

Отключение

  • Нажмите кнопку Пуск, щелкните стрелку рядом с командой Выход из системы и выберите команду Отключение.
