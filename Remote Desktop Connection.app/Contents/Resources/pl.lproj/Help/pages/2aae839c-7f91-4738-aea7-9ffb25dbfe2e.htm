Nie mogę kopiować ani wklejać między komputerami

Pokaż wszystko

Przyczyna: Niektórych formatów schowka nie można wklejać.

Rozwiązanie: Należy skopiować tekst do programu pośredniego, który obsługuje zgodny format schowka.

Kopiowanie z komputera z systemem Windows i wklejanie na komputerze Mac

  1. Na komputerze z systemem Windows skopiuj wybrany tekst.

  2. Na komputerze Mac otwórz program TextEdit i wklej tekst.

  3. Skopiuj tekst z programu TextEdit, a następnie wklej go do odpowiedniego pliku.

Kopiowanie z komputera Mac i wklejanie na komputerze z systemem Windows

  1. Na komputerze Mac skopiuj wybrany tekst.

  2. Na komputerze z systemem Windows otwórz Notatnik lub program Wordpad, a następnie wklej tekst.

  3. Skopiuj tekst z Notatnika lub programu Wordpad, a następnie wklej go do odpowiedniego pliku.

Przyczyna: Plik jest niedostępny z komputera z systemem Windows.

Rozwiązanie: Przed uruchomieniem sesji pulpitu zdalnego należy udostępnić dyski komputera Mac komputerowi z systemem Windows.

  1. W menu RDC kliknij polecenie Preferencje.

  2. Kliknij pozycję Dyski, a następnie wybierz z menu podręcznego odpowiednią opcję.

    Dysk lub folder jest teraz dostępny, dzięki czemu można korzystać z plików na komputerze Mac z poziomu komputera z systemem Windows.

    Jeśli ta zmiana zostanie wprowadzona po nawiązaniu połączenia, wybrany dysk lub folder stanie się dostępny po następnym nawiązaniu połączenia.

  3. W menu Start kliknij polecenie Komputer lub Mój komputer.

    Udostępniony dysk lub folder na komputerze Mac jest widoczny obok innych dysków i folderów na komputerze z systemem Windows.

    Uwaga  Jeśli menu Start nie jest widoczne, w menu Widok kliknij polecenie Pełny ekran.Aby wrócić później do trybu okna, wskaż górną krawędź ekranu, aż zostanie wyświetlony pasek menu RDC. Na pasku menu kliknij pozycję Widok, a następnie wyczyść pole wyboru Pełny ekran.

  4. Otwórz dysk lub folder źródłowy kopiowania.

  5. W menu Start kliknij polecenie Komputer lub Mój komputer.

  6. Otwórz dysk lub folder docelowy kopiowania.

    Teraz powinny być widoczne dwa otwarte okna: jedno odpowiadające lokalizacji źródłowej, a drugie — lokalizacji docelowej kopiowania.

  7. Przeciągnij do odpowiedniej lokalizacji dokument, który chcesz skopiować.

Przyczyna: Zasady grupy usług terminalowych nie zezwalają na udostępnianie dysków.

Rozwiązanie: Skontaktuj się z administratorem sieci w sprawie zasad grupy usług terminalowych.

Zasady grupy zastępują wszelkie preferencje ustawione za pomocą usługi Podłączanie pulpitu zdalnego. Gdy zasady grupy usług terminalowych uniemożliwiają udostępnianie dysków, nie można udostępniać plików między komputerami.

Przyczyna: Zasady grupy usług terminalowych nie zezwalają na udostępnianie schowka.

Rozwiązanie: Skontaktuj się z administratorem sieci w sprawie zasad grupy usług terminalowych.

Zasady grupy zastępują wszelkie preferencje ustawione za pomocą usługi Podłączanie pulpitu zdalnego. Gdy zasady grupy usług terminalowych uniemożliwiają udostępnianie schowka, nie można kopiować ani wklejać między komputerami.
