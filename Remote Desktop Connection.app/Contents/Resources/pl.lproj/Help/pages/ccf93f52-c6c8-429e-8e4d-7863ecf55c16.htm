Opcje dostępu uniwersalnego

Pokaż wszystko

Usługa Podłączanie pulpitu zdalnego korzysta z kilku wybranych funkcji dostępu uniwersalnego systemu Mac OS X. Funkcje dostępu uniwersalnego ułatwiają korzystanie z komputera użytkownikom, którzy mają słaby wzrok lub słuch albo ograniczoną sprawność ruchową.

Aby ustawić opcje dostępu, należy w menu Apple kliknąć pozycję Preferencje systemowe, a następnie w obszarze System kliknąć pozycję Dostęp uniwersalny. W poniższej tabeli znajduje się lista funkcji dostępu uniwersalnego systemu Mac OS X, z których można korzystać w usłudze Podłączanie pulpitu zdalnego.

Funkcje systemu Mac OS X nie działają na pulpicie systemu Windows po ustanowieniu sesji pulpitu zdalnego. Informacje na temat funkcji ułatwień dostępu w systemie Windows można znaleźć na stronie sieci Web Ułatwienia dostępu w produktach firmy Microsoft pod adresem http://www.microsoft.com/enable.

VoiceOver

Możliwość usłyszenia przez użytkownika nazw większości poleceń menu, okien dialogowych i innych elementów w usłudze Podłączanie pulpitu zdalnego. Funkcja nie obsługuje zawartości okna sesji pulpitu zdalnego. Na przykład po nawiązaniu połączenia z komputerem z systemem Windows funkcja VoiceOver nie odczytuje żadnych elementów znajdujących się na pulpicie systemu Windows.

Pełny dostęp klawiatury

Korzystanie z klawiatury zamiast myszy do poruszania się między formantami.

Uwaga  Opisy kart dotyczą amerykańskiego układu klawiatury. Klawisze w innych układach nie muszą być takie same jak w układzie amerykańskim. Różnice mogą również dotyczyć skrótów klawiaturowych używanych w komputerach przenośnych.

Dodatkowe skróty klawiaturowe można znaleźć w Pomocy systemu Mac.
