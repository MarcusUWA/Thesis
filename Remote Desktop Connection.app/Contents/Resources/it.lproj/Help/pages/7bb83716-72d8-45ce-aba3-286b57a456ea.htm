Connessione a una sessione console server

Mostra tutto
Importante

Per accedere a una sessione console su un server remoto che esegue Windows Server, è necessario essere amministratori del computer Windows.

  1. Aprire Connessione desktop remoto.

  2. Nella casella Computer, digitare il nome o l'indirizzo IP del computer Windows a cui si desidera connettersi, seguito da /console (ad esempio, FabricamFileServer/console).

  3. Fare clic su Connetti.

  4. Digitare il nome utente, la password e il dominio (se necessario).

Nota  Questo metodo di accesso funziona solo quando ci si connette a un server che esegue Windows Server.
