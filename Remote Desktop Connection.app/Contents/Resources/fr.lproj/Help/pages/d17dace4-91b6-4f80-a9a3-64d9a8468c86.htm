Connexion à plusieurs ordinateurs

Tout afficher

Si vous voulez vous connecter simultanément à plusieurs ordinateurs Windows, vous pouvez enregistrer un fichier de connexion pour chacun d'eux. Chaque fichier de connexion peut avoir son propre jeu de préférences, telles que des informations d'identification de connexion automatique, une taille d'écran et des options d'affichage. Vous pouvez rapidement vous connecter à l'ordinateur souhaité en ouvrant son fichier de connexion.

Important

L'ordinateur Windows doit être allumé et disposer d'une connexion réseau. Vous ne pouvez pas établir une connexion avec un ordinateur en état de veille ou de veille prolongée. Assurez-vous donc que les paramètres de veille et de veille prolongée de l'ordinateur Windows sont définis sur Jamais. Pour plus d'informations sur l'apport de ces modifications, voir l'Aide de Windows. Assurez-vous que l'ordinateur auquel vous voulez vous connecter exécute une version de Windows prenant en charge les connexions à distance. Pour obtenir une liste complète, voir Compatibilité entre les versions de Windows.

Étape 1 : connexion au premier ordinateur Windows

  1. Sur l'ordinateur Macintosh, ouvrez Connexion Bureau à Distance.

  2. Dans le champ Ordinateur, tapez le nom de l'ordinateur Windows auquel vous voulez vous connecter, puis cliquez sur Connecter.

  3. Tapez vos nom d'utilisateur, mot de passe et domaine pour l'ordinateur Windows si nécessaire.

Étape 2 : enregistrement d'un fichier de connexion pour le premier ordinateur Windows

Sélectionnez vos préférences de connexion pour l'ordinateur auquel vous êtes connecté.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Sélectionnez les options souhaitées.

  3. Dans le menu Fichier, cliquez sur Enregistrer sous.

  4. Dans le champ Enregistrer sous, tapez le nom de votre fichier de connexion.

  5. Cliquez sur Enregistrer.

Conseil  Pour vous connecter rapidement à un ordinateur Windows, double-cliquez sur le fichier de connexion ou faites glisser le fichier sur l'icône Connexion Bureau à Distance. Les fichiers de connexion sont enregistrés dans Documents/RDC Connections.

Étape 3 : connexion à un autre ordinateur Windows

  1. Dans la barre de menus de Connexion Bureau à Distance, cliquez sur Fichier, puis sur Nouvelle connexion.

  2. Dans le champ Ordinateur, tapez le nom de l'ordinateur Windows auquel vous voulez vous connecter, puis cliquez sur Connecter.

  3. Tapez vos nom d'utilisateur, mot de passe et domaine pour l'ordinateur Windows si nécessaire.

    Deux fenêtres de session sont à présent ouvertes sur votre bureau Mac, une pour chaque ordinateur Windows auquel vous êtes connecté. Pour ouvrir des connexions supplémentaires, répétez cette étape.

Étape 4 : enregistrement d'un fichier de connexion pour un autre ordinateur Windows

  1. Sélectionnez la fenêtre de session de l'ordinateur Windows supplémentaire auquel vous êtes connecté.

  2. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  3. Sélectionnez les options souhaitées.

  4. Dans le menu Fichier, cliquez sur Enregistrer sous.

  5. Dans le champ Enregistrer sous, tapez le nom de votre fichier de connexion.

  6. Cliquez sur Enregistrer.

  7. Répétez cette procédure pour chaque ordinateur Windows supplémentaire auquel vous êtes connecté.

Conseil  Pour vous connecter rapidement à un ordinateur Windows, double-cliquez sur le fichier de connexion ou faites glisser le fichier sur l'icône Connexion Bureau à Distance. Les fichiers de connexion sont enregistrés dans Documents/RDC Connections.
