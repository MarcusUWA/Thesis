Nie mogę nawiązać połączenia z komputerem z systemem Windows

Pokaż wszystko

Przyczyna: Komputer z systemem Windows nie jest skonfigurowany do akceptowania połączeń zdalnych.

Rozwiązanie: Należy włączyć ustawienie pulpitu zdalnego na komputerze, z którym ma zostać nawiązane połączenie.

Włączanie ustawienia pulpitu zdalnego w systemie Windows 7

Ważne

Aby zmieniać ustawienia pulpitu zdalnego, trzeba być administratorem komputera z systemem Windows.

  1. Na komputerze z systemem Windows kliknij przycisk Start , a następnie kliknij polecenie Panel sterowania.

  2. Kliknij pozycję System i zabezpieczenia, a następnie w obszarze System kliknij opcję Zezwalaj na dostęp zdalny.

  3. W obszarze Pulpit zdalny kliknij odpowiednią opcję.

    Jeśli zostanie wyświetlony monit o podanie hasła administratora lub o potwierdzenie, wpisz hasło lub potwierdź.

    Uwaga  Uwierzytelnianie na poziomie sieci poprawia bezpieczeństwo, ponieważ przed zainicjowaniem połączenia zdalnego wymaga wprowadzenia informacji logowania z systemu Windows na komputerze Mac. Więcej informacji na temat uwierzytelniania na poziomie sieci można znaleźć w Pomocy systemu Windows.

  4. Kliknij przycisk Wybierz użytkowników.

  5. Wykonaj dowolną z następujących czynności:

    Efekt

    Wykonaj te czynności

    Sprawdzanie dostępu

    Upewnij się, że Twoje konto użytkownika jest wymienione pod polem. Konto administratora komputera jest tam umieszczane automatycznie.

    Dodawanie kolejnego konta użytkownika

    Kliknij przycisk Dodaj, a następnie wpisz w polu odpowiednią nazwę konta użytkownika.

  3. Na karcie Zdalny zaznacz pole wyboru Zezwalaj użytkownikom na zdalne łączenie się z tym komputerem.

  4. Kliknij przycisk Wybierz użytkowników zdalnych.

Rozwiązanie: Należy się upewnić, że Zapora systemu Windows zezwala na wyjątki dotyczące pulpitu zdalnego.

Zezwalanie na wyjątki Zapory systemu Windows na komputerze z systemem Windows 7

Ważne

Aby zmieniać ustawienia zapory, trzeba być administratorem komputera z systemem Windows.

  1. Na komputerze z systemem Windows kliknij przycisk Start , a następnie kliknij polecenie Panel sterowania.

  2. Kliknij pozycję System i zabezpieczenia, a następnie w obszarze Zapora systemu Windows kliknij opcję Zezwalaj programowi na dostęp przez Zaporę systemu Windows.

  3. Na liście Dozwolone programy i funkcje upewnij się, że pole wyboru Pulpit zdalny jest zaznaczone.

Zezwalanie na wyjątki Zapory systemu Windows na komputerze z systemem Windows XP

Ważne

Aby zmieniać ustawienia zapory, trzeba być administratorem komputera z systemem Windows.

  1. Na komputerze z systemem Windows kliknij przycisk Start, a następnie kliknij polecenie Panel sterowania.

    Jeśli pracujesz w widoku klasycznym, w okienku nawigacji kliknij pozycję Przełącz do widoku kategorii.

  2. Kliknij pozycję Centrum zabezpieczeń, a następnie w obszarze Zarządzaj ustawieniami zabezpieczeń dla kliknij pozycję Zapora systemu Windows.

  3. Upewnij się, że pole wyboru Nie zezwalaj na wyjątki jest wyczyszczone.

  4. Kliknij kartę Wyjątki i upewnij się, że pole wyboru Pulpit zdalny jest zaznaczone.

Przyczyna: Komputer Mac nie może nawiązać połączenia z siecią.

Rozwiązanie: Należy się upewnić, że komputer z systemem Windows jest włączony i połączony z siecią.

Połączenie zdalne może się zakończyć niepowodzeniem z powodu problemów z siecią, takich jak wyłączenie sieci zdalnej lub odłączenie kabla Ethernet komputera z systemem Windows.

Przyczyna: Połączenia są blokowane przez zasady sieci.

Rozwiązanie: Należy skontaktować się z administratorem sieci w sprawie zasad zabezpieczeń organizacji.

Zasady zabezpieczeń zależą od organizacji. Aby na przykład nawiązać połączenie z komputerem z systemem Windows w sieci, w której do zabezpieczania ruchu sieciowego jest używany protokół IPsec, należy zdefiniować dla tego komputera wyjątek graniczny.

Przyczyna: Nie można znaleźć komputera z systemem Windows.

Rozwiązanie: Należy się upewnić, że ustawienia sieci na komputerze Mac są poprawnie skonfigurowane do obsługi typu używanego połączenia.

Ustawienia połączenia z siecią Ethernet lub połączenia bezprzewodowego na komputerze Mac muszą być skonfigurowane dla sieci, z którą ma zostać nawiązane połączenie.

Rozwiązanie: Należy się upewnić, że wprowadzono poprawną nazwę komputera.

Sprawdzanie nazwy komputera w systemie Windows 7

  1. Na komputerze z systemem Windows kliknij przycisk Start , a następnie kliknij polecenie Panel sterowania.

  2. Kliknij pozycję System i zabezpieczenia, a następnie w obszarze System kliknij opcję Wyświetl nazwę tego komputera.

  3. W obszarze Nazwa komputera, domena i ustawienia grupy roboczej zanotuj nazwę komputera lub pełną nazwę komputera, jeśli komputer z systemem Windows należy do domeny.

    Po otwarciu usługi Podłączanie pulpitu zdalnego na komputerze Mac należy wprowadzić tę nazwę.

Sprawdzanie nazwy komputera w systemie Windows Vista

  1. Na komputerze z systemem Windows kliknij przycisk Start , a następnie kliknij polecenie Panel sterowania.

    Jeśli pracujesz w widoku klasycznym, w okienku nawigacji kliknij pozycję Strona główna Panelu sterowania.

  2. Kliknij pozycję System i konserwacja, a następnie kliknij pozycję System.

  3. W obszarze Nazwa komputera, domena i ustawienia grupy roboczej zanotuj nazwę komputera lub pełną nazwę komputera, jeśli komputer z systemem Windows należy do domeny.

    Po otwarciu usługi Podłączanie pulpitu zdalnego na komputerze Mac należy wprowadzić tę nazwę.

Sprawdzanie nazwy komputera w systemie Windows XP

  1. Na komputerze z systemem Windows kliknij przycisk Start, kliknij prawym przyciskiem myszy pozycję Mój komputer, a następnie kliknij polecenie Właściwości.

  2. Na karcie Nazwa komputera zanotuj pełną nazwę komputera.

    Po otwarciu usługi Podłączanie pulpitu zdalnego na komputerze Mac należy wprowadzić tę nazwę.

Rozwiązanie: Jeśli nie można nawiązać połączenia przy użyciu nazwy komputera, należy użyć jego adresu IP.

  1. Na komputerze z systemem Windows kliknij przycisk Start, kliknij pozycję Wszystkie programy, kliknij pozycję Akcesoria, a następnie kliknij pozycję Wiersz polecenia.

  2. W wierszu polecenia wpisz ciąg ipconfig, a następnie naciśnij klawisz ENTER.

  3. Zanotuj adres IP komputera wyświetlany w obszarze Karta Ethernet Połączenie lokalne.

    Po otwarciu usługi Podłączanie pulpitu zdalnego na komputerze Mac można wprowadzić ten adres IP zamiast nazwy komputera. Na komputerach z systemami Windows Vista lub Windows 7 adres IP jest podawany jako „adres IPv4”.

Rozwiązanie: Należy się upewnić, że połączenie internetowe na komputerze Mac działa.

  • Otwórz przeglądarkę sieci Web (na przykład Safari) i upewnij się, że wyświetlenie strony sieci Web jest możliwe.

Przyczyna: Nie można zweryfikować tożsamości komputera z systemem Windows.

Rozwiązanie: Należy się upewnić, że preferencje zabezpieczeń usługi Podłączanie pulpitu zdalnego są zgodne z wersją systemu Windows na komputerze z systemem Windows.

Jeśli na komputerze z systemem Windows jest uruchomiony system Windows XP lub starsza wersja albo system Windows Server 2003 z dodatkiem Service Pack 1 (SP1) lub starsza wersja, komputer nie może przedstawić swojej tożsamości do zweryfikowania. Jeśli na komputerze zdalnym jest uruchomiona jedna z podanych wersji systemu Windows, a w preferencjach zabezpieczeń na komputerze Mac do nawiązania połączenia jest wymagane uwierzytelnienie, połączenie zdalne zakończy się niepowodzeniem.

  1. W menu RDC kliknij pozycję Preferencje, a następnie kliknij pozycję Zabezpieczenia.

  2. Upewnij się, że opcja Nie łącz w przypadku niepowodzenia uwierzytelniania nie jest zaznaczona.
