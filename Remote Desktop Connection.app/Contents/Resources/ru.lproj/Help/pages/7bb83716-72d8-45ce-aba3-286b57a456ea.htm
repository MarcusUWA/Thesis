Подключение к сеансу консоли сервера

Показать все
Важно

Чтобы войти в сеанс консоли удаленного сервера с операционной системой Windows Server, необходимо иметь разрешения администратора на компьютере под управлением Windows.

  1. Откройте подключение к удаленному рабочему столу.

  2. В поле Компьютер введите имя или IP-адрес компьютера под управлением Windows, к которому требуется подключиться, указав после него параметр /console (например, FabricamFileServer/console).

  3. Выберите команду Подключение.

  4. Введите имя пользователя, пароль и домен (если требуется).

Примечание  Этот способ входа работает только при подключении к серверу с ОС Windows Server.
