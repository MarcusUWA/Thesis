Tastenkombinationen

Alles anzeigen

Mit Tastenkombinationen, d. h. durch Drücken einer oder mehrerer Tasten auf der Tastatur, können Sie häufig wiederkehrende Aufgaben schnell und effizient ausführen.

Die beschriebenen Tastenkombinationen beziehen sich auf das US-amerikanische Tastaturlayout. Die Tasten anderer Tastaturlayouts weichen möglicherweise davon ab. Die Tastenkombinationen auf Laptops können auch anders sein.

Bearbeiten und Formatieren

Wenn Sie auf dem Windows-Desktop arbeiten, können Sie die gleichen Aufgaben mit den entsprechenden Windows-Tastenkombinationen ausführen. Wenn Sie Text aus einem Dokument auf Ihrem Windows-Desktop kopieren möchten, drücken Sie beispielsweise STRG+C oder ⌘+C. Wenn Sie den Inhalt in ein Dokument auf dem Macintosh einfügen möchten, drücken Sie ⌘+V.

Die Remotedesktopverbindung verfügt über Standardtastenbelegungen, die Aufgaben auf dem Windows-basierten Computer entsprechen.

Die nachfolgend aufgeführten Tastaturzuweisungen können deaktiviert oder geändert werden.Klicken Sie im Menü RDC auf Einstellungen und auf Tastatur. Doppelklicken Sie in der Spalte Mac auf die Taste, die Sie ändern möchten, und geben Sie die gewünschte Tastenkombination ein.
