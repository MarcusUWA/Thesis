подключение к удаленному рабочему столу сохраняет параметры подключения в файле подключения по умолчанию. Чтобы тратить меньше времени на подключение, можно сохранить дополнительные имена компьютеров и параметры подключений для каждого компьютера под управлением Windows, с которым устанавливается соединение. Например, можно открыть доступ ко всем дискам Mac при подключении к одному компьютеру под управлением Windows, а для другого компьютера под управлением Windows открыть доступ только к одной папке на компьютере Mac. Кроме того, можно сохранить несколько файлов подключений для одного компьютера под управлением Windows.

Выполните одно из перечисленных ниже действий.

Создание файла подключения

  1. В меню RDC выберите пункт Настройки.

    Примечание  Чтобы перейти к настройке параметров, не обязательно вводить данные пользователя в области входа.

  2. Установите для этого подключения нужные параметры, например размер экрана, автоматический ввод учетных данных и параметры безопасности.

  3. В меню Файл выберите команду Сохранить как.

  4. Введите имя файла подключения и нажмите кнопку Сохранить.

    При установке подключения выбранные параметры станут параметрами по умолчанию для всех подключений к удаленному рабочему столу.

Совет  Чтобы быстро подключиться к компьютеру под управлением Windows, дважды щелкните файл подключения или перетащите его на значок подключение к удаленному рабочему столу. Файлы подключений хранятся в папке Documents/RDC Connections.

Изменение файла подключения

  1. В подключение к удаленному рабочему столу в меню Файл выберите команду Изменить подключение.

  2. Выберите нужный файл подключения и нажмите кнопку Открыть.

  3. Внесите необходимые изменения.

  4. В меню Файл выберите команду Сохранить.

    При установке подключения выбранные параметры станут параметрами по умолчанию для всех подключений к удаленному рабочему столу.

    Совет  Кроме того, можно изменить параметры подключения непосредственно во время сеанса. В меню RDC выберите пункт Настройки и внесите нужные изменения. Они вступят в силу при следующем подключении.
