Kopiowanie plików między komputerami

Pokaż wszystko

Aby skopiować pliki, należy najpierw otworzyć dysk lub folder komputera Mac na komputerze z systemem Windows, a następnie skopiować odpowiedni plik na komputer z systemem Windows. Nie można przeciągnąć pliku z komputera Mac na komputer z systemem Windows.

Krok 1. Udostępnianie dysku lub folderu komputera Mac komputerowi z systemem Windows

  1. W menu RDC kliknij polecenie Preferencje.

  2. Kliknij pozycję Dyski, a następnie wybierz z menu podręcznego odpowiednią opcję.

Dysk lub folder jest teraz dostępny, dzięki czemu można korzystać z plików na komputerze Mac z poziomu komputera z systemem Windows.

Krok 2. Kopiowanie plików między komputerami

Po udostępnieniu używanego obecnie dysku lub folderu komputera Mac komputerowi z systemem Windows można kopiować pliki między komputerami.

  1. W menu Start kliknij polecenie Komputer lub Mój komputer.

    Udostępniony dysk lub folder na komputerze Mac jest widoczny obok innych dysków i folderów na komputerze z systemem Windows.

  2. Otwórz dysk lub folder źródłowy kopiowania.

  3. W menu Start kliknij polecenie Komputer lub Mój komputer.

  4. Otwórz dysk lub folder docelowy kopiowania.

    Teraz powinny być widoczne dwa otwarte okna: jedno odpowiadające lokalizacji źródłowej, a drugie — lokalizacji docelowej kopiowania.

  5. Przeciągnij do odpowiedniej lokalizacji dokument, który chcesz skopiować.
