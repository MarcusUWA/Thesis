Print a document

Show All

You can print documents that are on your Windows-based computer to a printer that is connected to your Mac. To do this, you first have to share the printer with the Windows-based computer.

After you make the printer available, the printer can be accessed on the Windows-based computer. If you save the connection fileA file that contains all of the information for a remote connection. For example, you can set the screen size for the session window, or share Mac drives with the Windows-based computer by saving these preferences to a connection file. If you connect to multiple Windows-based computers, you can save a different connection file for each computer. and log in with that connection file during later remote sessions, the printer remains available.

Do any of the following:

Make a printer available to the Windows-based computer

  1. On the RDC menu, click Preferences.

  2. Click Printers, and then select the Use a printer that is connected to the Mac check box.

  3. Select the printer that you want.

    Tips 
     

    • If you want to select a printer at the time that you print, select All printers.

    • To save a connection file that includes access to the printer, on the File menu, click Save As, and then type a name for the connection file.

Print a Windows-based document to a printer that is connected to a Mac

  1. On the Windows desktop, open and print the document that you want to print.

  2. Select the printer that you want to print to, and then click OK.
