Connexion à une session de console du serveur

Tout afficher
Important

Pour vous connecter à une session de console sur un serveur distant exécutant Windows Server, vous devez être administrateur sur l'ordinateur Windows.

  1. Ouvrez Connexion Bureau à Distance.

  2. Dans le champ Ordinateur, tapez le nom ou l'adresse IP de l'ordinateur Windows auquel vous voulez vous connecter, suivi de /console (par exemple, FabricamFileServer/console).

  3. Cliquez sur Connecter.

  4. Tapez vos nom d'utilisateur, mot de passe et domaine (si nécessaire).

Remarque  Cette méthode de connexion ne fonctionne que si vous vous connectez à un serveur exécutant Windows Server.
