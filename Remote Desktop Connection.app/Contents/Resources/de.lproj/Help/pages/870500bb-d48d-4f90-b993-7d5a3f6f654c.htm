Neuigkeiten hinsichtlich der Remotedesktopverbindung

Alles anzeigen

Die Microsoft Remotedesktopverbindungs-Client für Mac 2 verfügt über eine neu gestaltete Benutzeroberfläche, verbesserte Anpassungsmöglichkeiten sowie die Funktion zum gleichzeitigen Herstellen mehrerer Verbindungen mit Windows-basierten Computern. Die folgende Tabelle enthält eine Liste dieser Verbesserungen.

Feature

Beschreibung

Universal Binaries

Ausführung auf Intel-basierten und PowerPC-basierten Macintosh-Computern möglich.

Benutzeroberfläche und Menüleiste

Vereinfachtes Verbindungsfenster und durch Menüs für häufige Aufgaben (Datei, Bearbeiten, Ansicht und Hilfe) erweiterte Menüleiste. Die Anwendungseinstellungen sind nun im Menü RDC zu finden.

Remote Desktop Protocol 6.0

Verbesserte Leistung und Kompatibilität mit Windows Vista und Windows 7

Mehrere Sitzungen

Ermöglicht das Herstellen einer Verbindung mit mehreren Computern, wenn eine Verbindungsdatei für jeden Computer gespeichert wird. Mit dem neuen Menü Datei können Sie neue, gespeicherte und zu einem früheren Zeitpunkt geöffnete Verbindungen öffnen und alle Änderungen an Verbindungsdateien verwalten.

Verbesserte Anpassungsmöglichkeiten gegenüber Remotedesktopverbindung 1.0

Akzeptiert Änderungen an Anwendungseinstellungen wie Tastenkombinationen während einer Remotedesktopsitzung. Diese Änderungen werden beim nächsten Herstellen einer Verbindung wirksam.

Automatisches Wiederherstellen der Verbindung

Stellt eine Remotesitzung wieder her, wenn die Netzwerkverbindung unterbrochen wurde.

Anpassung der Bildschirmgröße

Führt die Remotedesktopsitzung in einem Fenster aus, dessen Größe angepasst werden kann, oder im Präsentationsmodus.

Drucken

Druckt Windows-basierte Dokumente auf jedem Drucker, der an den Macintosh-Computer angeschlossen ist.

Authentifizierung auf Netzwerkebene

Überprüft die Identität des Windows-basierten Computers vor dem Herstellen einer Remotedesktopverbindung. Sie können diese Option auswählen, wenn Sie eine Verbindung mit einem Computer unter Windows Vista oder Windows 7 herstellen. Die Authentifizierung auf Netzwerkebene ist sicherer als andere Authentifizierungsoptionen, die in früheren Versionen von Windows verwendet wurden.

Breitbildauflösung

Ermöglicht eine verbesserte Auflösung im Breitbildformat durch entsprechende Einstellungen.

Serverkonsolensitzungen

Beschleunigt die Anmeldung, wenn Sie "console" als Teil des Computernamens angeben.
