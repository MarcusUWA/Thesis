Raccourcis clavier

Tout afficher

Les raccourcis clavier (une ou plusieurs touches sur lesquelles vous appuyez pour effectuer une tâche) permettent d'effectuer des tâches rapidement.

Les descriptions des raccourcis clavier font référence au modèle de clavier américain. Il se peut que les touches des autres modèles de clavier ne correspondent pas aux touches du clavier américain. Par ailleurs, les raccourcis clavier d'un ordinateur portable peuvent également être différents.

Modification et mise en forme

Lorsque vous travaillez sur le bureau Windows, vous pouvez exécuter les mêmes tâches à l'aide des raccourcis clavier Windows correspondants. Par exemple, pour copier du texte d'un document sur votre bureau Windows, vous pouvez appuyer sur les touches CONTROL+C ou ⌘+C. Pour copier le contenu dans un document sur votre ordinateur Mac, appuyez sur ⌘+V.

Utilisation de raccourcis clavier Mac sur le bureau Windows

Connexion Bureau à Distance inclut des affectations de touche par défaut correspondant à des tâches sur l'ordinateur Windows.

Vous pouvez désactiver ou modifier n'importe laquelle des affectations de touches suivantes. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences, puis sur Clavier . Dans la colonne Mac, double-cliquez sur la touche à modifier, puis entrez le raccourci clavier de votre choix.
