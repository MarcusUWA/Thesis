Настройка компьютера под управлением Windows для удаленного доступа

Показать все

Прежде чем подключаться к компьютеру под управлением Windows с помощью подключение к удаленному рабочему столу, необходимо разрешить на нем удаленные подключения.

Важно

Компьютер под управлением Windows должен быть включен и подключен к сети. Невозможно подключиться к компьютеру, который находится в спящем или ждущем режиме либо в режиме гибернации. Поэтому на компьютере под управлением Windows для всех этих режимов должен быть установлен параметр "Никогда". Сведения о том, как это сделать, см. в справке Windows. На компьютере, к которому требуется подключиться, должна быть установлена версия Windows с поддержкой удаленных подключений. Полный список см. в статье Совместимость версий Windows.

Выполните одно из перечисленных ниже действий.

Разрешение удаленных подключений на компьютере под управлением Windows 7

Важно

Изменять параметры удаленного рабочего стола может только администратор компьютера под управлением Windows.

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

  2. Щелкните элемент Система и безопасность, затем — Система и выберите пункт Настройка удаленного доступа.

  3. В разделе Удаленный рабочий стол выберите нужный параметр.

    При запросе пароля администратора или его подтверждения введите пароль или подтвердите его.

    Примечание  Проверка подлинности на уровне сети повышает уровень безопасности, поскольку перед открытием удаленного подключения на компьютере Mac требуется ввести регистрационные данные Windows. Дополнительные сведения о проверке подлинности на уровне сети см. в справке Windows.

  4. Нажмите кнопку Выбрать пользователей.

  5. Выполните любое из указанных ниже действий.

    Чтобы

    Выполните следующие действия

    Проверить доступ

    Проверьте, указана ли ваша учетная запись пользователя в списке под полем. Учетная запись администратора компьютера помещается в список автоматически.

    Добавить другую учетную запись пользователя

    Нажмите кнопку Добавить и введите нужное имя учетной записи в поле.

  4. Нажмите кнопку Выбрать пользователей.

  5. Выполните любое из указанных ниже действий.
