Mac のハード ディスクにアクセスできない

すべて表示

原因: Mac のハード ディスクが Windows コンピューターと共有されていません。

対処方法: Windows コンピューターに公開する Mac のディスクまたはフォルダーを選択します。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [ドライブ] をクリックし、ポップアップ メニューで必要なオプションを選択します。

    ディスクまたはフォルダーが利用可能になり、Windows ベースのコンピューターから Mac 上のファイルにアクセスできるようになります。

  3. Windows コンピューターで [ [スタート] ] をクリックし、[ [コンピューター] ] または [ [マイ コンピュータ] ] をクリックします。

    利用可能になった Mac のディスクまたはフォルダーが、Windows ベースのコンピューター上の他のすべてのディスクやフォルダーと共に一覧に表示されます。

原因: ターミナル サービスのグループ ポリシーでディスク共有が許可されていません。

対処方法: ターミナル サービスのグループ ポリシーの詳細については、ネットワーク管理者に問い合わせてください。

グループ ポリシーは、リモート デスクトップ接続 のどの環境設定よりも優先されます。 ターミナル サービスのグループ ポリシーでディスク共有が禁止されているときは、Windows コンピューター上で Mac のディスクを参照できません。
