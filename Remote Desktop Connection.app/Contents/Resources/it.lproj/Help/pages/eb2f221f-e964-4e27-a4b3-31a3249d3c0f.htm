Stampa di un documento

Mostra tutto

È possibile stampare documenti presenti su computer Windows su una stampante connessa al Mac. Per eseguire questa operazione, è innanzitutto necessario condividere la stampante con il computer Windows.

Non appena la stampante sarà disponibile, sarà possibile accedervi dal computer Windows. Se si salva il file di connessioneFile contenente tutte le informazioni per la connessione remota. Ad esempio, è possibile impostare le dimensioni dello schermo per la finestra della sessione o condividere unità Mac con il computer Windows salvando queste preferenze in un file di connessione. Se ci si connette a più computer Windows, è possibile salvare un file di connessione diverso per ciascun computer. e si accede con tale file di connessione durante le sessioni remote successive, la stampante risulterà disponibile.

Eseguire una delle seguenti operazioni:

Rendere disponibile una stampante sul computer Windows

  1. Scegliere Preferenze dal menu Connessione desktop remoto.

  2. Fare clic su Stampanti, quindi selezionare la casella di controllo Utilizza una stampante connessa al Mac.

  3. Selezionare la stampante desiderata.

    Suggerimenti 
     

    • Se si desidera selezionare una stampante al momento della stampa, selezionare Tutte le stampanti.

    • Per salvare un file di connessione che include l'accesso alla stampante, scegliere Salva con nome dal menu File, quindi digitare un nome per il file di connessione.

Stampa di un documento Windows su una stampante connessa a un Mac

  1. Sul desktop di Windows aprire e stampare il documento desiderato.

  2. Selezionare la stampante da cui eseguire la stampa e fare clic su OK.
