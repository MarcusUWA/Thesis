Подключение к компьютеру под управлением Windows

Показать все

Перед подключением необходимо разрешить доступ к удаленному рабочему столу на нужном компьютере. Ниже описана процедура подключения к компьютеру под управлением Windows с компьютера Mac.

Примечание  клиент подключения к удаленному рабочему столу (Майкрософт) для Mac 2 не поддерживает использование смарт-карт.

Действие 1. Разрешение удаленных подключений на компьютере под управлением Windows

Используйте одну из процедур, описанных ниже, в зависимости от того, какая версия операционной системы запущена на компьютере под управлением Windows, к которому требуется подключиться.

Разрешение удаленных подключений в ОС Windows 7

Важно

Изменять параметры удаленного рабочего стола может только администратор компьютера под управлением Windows.

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

  2. Щелкните элемент Система и безопасность, а затем в разделе Система выберите команду Настройка удаленного доступа.

  3. В разделе Удаленный рабочий стол выберите нужный параметр.

    При запросе пароля администратора или его подтверждения введите пароль или подтвердите его.

    Примечание  Проверка подлинности на уровне сети повышает уровень безопасности, поскольку перед открытием удаленного подключения на компьютере Mac требуется ввести регистрационные данные Windows. Дополнительные сведения о проверке подлинности на уровне сети см. в справке Windows.

  4. Нажмите кнопку Выбрать пользователей.

  5. Выполните любое из указанных ниже действий.

    Чтобы

    Выполните следующие действия

    Проверить доступ

    Проверьте, указана ли ваша учетная запись пользователя в списке под полем. Учетная запись администратора компьютера помещается в список автоматически.

    Добавить другую учетную запись пользователя

    Нажмите кнопку Добавить и введите нужное имя учетной записи в поле.

Действие 2. Поиск имени компьютера под управлением Windows

Открыв подключение к удаленному рабочему столу, необходимо указать имя компьютера под управлением Windows, к которому требуется подключиться. Используйте одну из процедур, описанных ниже, в зависимости от того, какая версия операционной системы запущена на компьютере под управлением Windows, к которому требуется подключиться.

Поиск имени компьютера в ОС Windows 7

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

  2. Щелкните элемент Система и безопасность, а затем в разделе Система выберите команду Просмотр имени этого компьютера.

  3. В разделе Имя компьютера, имя домена и параметры рабочей группы уточните имя компьютера или полное имя компьютера, если компьютер под управлением Windows является членом домена.

    При запуске подключение к удаленному рабочему столу на компьютере Mac необходимо ввести это имя.

Поиск имени компьютера в ОС Windows Vista

  1. На компьютере под управлением Windows нажмите кнопку Пуск и выберите пункт Панель управления.

    В классическом виде в области навигации выберите пункт Панель управления — домашняя страница.

  2. Щелкните элемент Система и ее обслуживание, а затем — Система.

  3. В разделе Имя компьютера, имя домена и параметры рабочей группы уточните имя компьютера или полное имя компьютера, если компьютер под управлением Windows является членом домена.

    При запуске подключение к удаленному рабочему столу на компьютере Mac необходимо ввести это имя.

Поиск имени компьютера в ОС Windows XP

  1. На компьютере под управлением Windows нажмите кнопку Пуск, щелкните правой кнопкой мыши элемент Мой компьютер и выберите пункт Свойства.

  2. На вкладке Имя компьютера уточните полное имя компьютера.

    При запуске подключение к удаленному рабочему столу на компьютере Mac необходимо ввести это имя.

Действие 3. Открытие нового подключения

Можно открыть подключение как к одному компьютеру под управлением Windows, так и к нескольким таким компьютерам одновременно.

Важно

Компьютер под управлением Windows должен быть включен и подключен к сети. Невозможно подключиться к компьютеру, который находится в спящем или ждущем режиме либо в режиме гибернации. Поэтому на компьютере под управлением Windows для всех этих режимов должен быть установлен параметр "Никогда". Сведения о том, как это сделать, см. в справке Windows. На компьютере, к которому требуется подключиться, должна быть установлена версия Windows с поддержкой удаленных подключений. Полный список см. в статье Совместимость версий Windows.

  1. На компьютере Macintosh откройте подключение к удаленному рабочему столу.

  2. В поле Компьютер введите имя компьютера под управлением Windows, к которому требуется подключиться, и выберите команду Подключение.

  3. При необходимости введите имя пользователя, пароль и домен для компьютера под управлением Windows.
