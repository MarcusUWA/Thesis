サーバー コンソール セッションに接続する

すべて表示
重要

Windows Server が動作しているリモート サーバー上のコンソール セッションにログインするには、Windows コンピューターの管理者権限が必要です。

  1. [Remote Desktop Connection] を開きます。

  2. [ [コンピューター] ] ボックスに、接続先の Windows コンピューターの名前または IP アドレスを入力し、続けて /console と入力します (例: FabricamFileServer/console)。

  3. [ [接続] ] をクリックします。

  4. ユーザー名、パスワード、およびドメイン (必要な場合) を入力します。

メモ  このログイン方法は、Windows Server が動作しているサーバーに接続するときにのみ有効です。
