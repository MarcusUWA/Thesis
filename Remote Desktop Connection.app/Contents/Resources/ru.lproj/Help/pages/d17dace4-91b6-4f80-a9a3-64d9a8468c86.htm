Подключение к нескольким компьютерам

Показать все

Если требуется подключаться сразу к нескольким компьютерам под управлением Windows, для каждого из них можно сохранить отдельный файл подключения. Каждому подключению может соответствовать собственный набор параметров, таких как учетные данные для автоматического входа, параметры размера экрана и отображения. Чтобы быстро подключиться к нужному компьютеру, достаточно просто открыть соответствующий файл подключения.

Важно

Компьютер под управлением Windows должен быть включен и подключен к сети. Невозможно подключиться к компьютеру, который находится в спящем или ждущем режиме либо в режиме гибернации. Поэтому на компьютере под управлением Windows для всех этих режимов должен быть установлен параметр "Никогда". Сведения о том, как это сделать, см. в справке Windows. На компьютере, к которому требуется подключиться, должна быть установлена версия Windows с поддержкой удаленных подключений. Полный список см. в статье Совместимость версий Windows.

Действие 1. Подключение к первому компьютеру под управлением Windows

  1. На компьютере Macintosh откройте подключение к удаленному рабочему столу.

  2. В поле Компьютер введите имя компьютера под управлением Windows, к которому требуется подключиться, и выберите команду Подключение.

  3. При необходимости введите имя пользователя, пароль и домен для компьютера под управлением Windows.

Действие 2. Сохранение файла подключения для первого компьютера под управлением Windows

Настройте параметры подключения для компьютера, с которым установлено соединение.

  1. В меню RDC выберите пункт Настройки.

  2. Установите нужные параметры.

  3. В меню Файл выберите команду Сохранить как.

  4. В поле Сохранить как укажите имя файла подключения.

  5. Нажмите кнопку Сохранить.

Совет  Чтобы быстро подключиться к компьютеру под управлением Windows, дважды щелкните файл подключения или перетащите его на значок подключение к удаленному рабочему столу. Файлы подключений хранятся в папке Documents/RDC Connections.

Действие 3. Подключение к другому компьютеру под управлением Windows

  1. В строке меню подключение к удаленному рабочему столу выберите команду Файл, а затем — пункт Создать подключение.

  2. В поле Компьютер введите имя компьютера под управлением Windows, к которому требуется подключиться, и выберите команду Подключение.

  3. При необходимости введите имя пользователя, пароль и домен для компьютера под управлением Windows.

    Теперь на рабочем столе Mac открыты два окна сеанса — по одному для каждого из компьютеров под управлением Windows, с которыми установлены соединения. Если требуется открыть дополнительные подключения, повторите это действие.

Действие 4. Сохранение файла подключения для второго компьютера под управлением Windows

  1. Выберите окно сеанса второго компьютера под управлением Windows, с которым установлено соединение.

  2. В меню RDC выберите пункт Настройки.

  3. Установите нужные параметры.

  4. В меню Файл выберите команду Сохранить как.

  5. В поле Сохранить как укажите имя файла подключения.

  6. Нажмите кнопку Сохранить.

  7. Повторите эту процедуру для каждого следующего компьютера под управлением Windows, с которым установлено соединение.

Совет  Чтобы быстро подключиться к компьютеру под управлением Windows, дважды щелкните файл подключения или перетащите его на значок подключение к удаленному рабочему столу. Файлы подключений хранятся в папке Documents/RDC Connections.
