Ograniczanie sesji do pojedynczej aplikacji systemu Windows

Pokaż wszystko

Jeśli użytkownik często pracuje w określonej aplikacji na serwerze z systemem Windows Server, może zaoszczędzić czas przez uruchomienie sesji pulpitu zdalnego wyłącznie z tą aplikacją. W przypadku połączeń komputerów klienckich z serwerem systemu Windows również administratorzy sesji mogą ograniczyć zakres tych połączeń do pojedynczej aplikacji. Gdy są wybrane te preferencje, pulpit systemu Windows nie jest widoczny ani dostępny.

  1. W menu RDC kliknij pozycję Preferencje, a następnie kliknij pozycję Aplikacje.

  2. Zaznacz pole wyboru.

  3. W polu Ścieżka aplikacji i nazwa pliku wpisz ścieżkę i nazwę pliku aplikacji, którą chcesz uruchomić.

    W razie potrzeby można również wpisać ścieżkę, której obiektem docelowym jest katalog roboczyFolder, w którym jest zainstalowana i uruchamiana aplikacja. Większość zainstalowanych aplikacji ma stały katalog roboczy i nie trzeba go zmieniać. Niektóre starsze aplikacje, programy narzędziowe oraz określone skrypty i pliki wsadowe są uruchamiane w innym folderze niż ten, w którym się znajdują. aplikacji.
