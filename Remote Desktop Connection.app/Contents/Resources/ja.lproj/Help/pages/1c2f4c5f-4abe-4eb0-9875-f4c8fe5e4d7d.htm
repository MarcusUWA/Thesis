すべて表示

アニメーションやグラフィックを多用する機能をネットワーク接続を介して転送するとき、Remote Desktop セッションのパフォーマンスが低下し、アプリケーションの動作が遅くなることがあります。セッションの開始後に Windows コンピューターで表示が変化すると、Macintosh コンピューターの側も更新されます。アニメーション処理は常に表示を更新する必要があるため、接続が低速な場合は特に、パフォーマンスを低下させる可能性があります。たとえば、アプリケーションや Web でビデオやアニメーション画像が使用されていると、この現象が起きることがあります。

次のようなオプションが原因で、セッションのパフォーマンスが遅くなる可能性があります。

  • デスクトップの背景を表示する

  • フォント スムージングを表示する

  • ドラッグ中にウィンドウの内容を表示する

  • メニューおよびウインドウ アニメーションを表示する

  • テーマを表示する

パフォーマンスを改善するために、これらのオプションの一部を無効にすることができます。

ヒント  表示オプションを無効にするには、[ [RDC] ] メニューの [ [環境設定] ] をクリックし、[ [表示] ] をクリックして、無効にするオプションのチェック ボックスをオフにします。
