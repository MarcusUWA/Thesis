Параметры отображения для подключения к удаленному рабочему столу

Показать все

Параметры отображения можно сохранять для каждого компьютера под управлением Windows, к которому устанавливается подключение. Основная часть этих параметров связана с отображением графических элементов. Выбор параметров может заметно повлиять на скорость работы сеанс удаленного рабочего столаПодключение между клиентским компьютером (Mac) и удаленным компьютером с системой Windows. Подключение может выполняться по локальной сети или через Интернет. При установлении сеанса удаленный рабочий стол компьютера с системой Windows отображается как окно на рабочем столе Mac. Сеанс завершается при отключении связи или выходе из системы Windows.. Значения параметров по умолчанию рассчитаны на оптимальную производительность при высокоскоростном подключении.

Параметр

Значение по умолчанию

Описание

Размер удаленного рабочего стола

1024 x 768

Изменение разрешения экрана и размера окна, в котором отображается сеанс подключения к удаленному рабочему столу. Например, если требуется, чтобы сеанс отображался в окне, размер которого меньше размера по умолчанию, уменьшите значение этого параметра.

Совет  Чтобы изменить размер окна во время сеанса, в меню Вид выберите пункт Вписать в окно и с помощью маркера изменения размера установите нужный размер окна.

Цвета

Тысячи

Установка глубины цвета для сеанса. Максимальная глубина цвета при удаленном подключении может быть ограничена политиками, которые используются в сети, и версией Windows на удаленном компьютере. Если выбранная глубина цвета превышает максимально допустимое значение, подключение к удаленному рабочему столу создаст подключение с максимально допустимой глубиной цвета.

Открывать окно удаленного рабочего стола на

Основной монитор

Запуск подключение к удаленному рабочему столу на выбранном мониторе. Если компьютер Mac настроен для использования двух и более мониторов, можно указать, на каком из них должно открываться окно сеанса.

Отображение фона рабочего стола

Выкл.

Отображение в окне сеанса фона или обоев рабочего стола Windows.

Отображение сглаживания шрифтов

Вкл.

Улучшение читаемости экранных шрифтов за счет сглаживания зубчатых диагональных и кривых линий в буквах.

Отображение содержимого окна при перетаскивании

Выкл.

Отображение всего содержимого окна при его перетаскивании по экрану (вместо отображения только рамки окна).

Отображение анимации меню и окон

Выкл.

Отображение вложенных меню и видеоэффектов для меню и всплывающих подсказок.

Отображение тем

Вкл.

Отображение в окне сеанса фона, звуков, значков и других элементов, настроенных на компьютере под управлением Windows.

Разрешить кэширование графики

Вкл.

Повышение производительности за счет уменьшения объема данных, пересылаемых через подключение.

Не показывать строку меню Mac и Dock в полноэкранном режиме

Выкл.

Запрет отображения строки меню и панели Dock, когда окно сеанса открыто в полноэкранном режиме. В частности, этот параметр позволяет избежать наложения панели Dock на панель задач Windows.
