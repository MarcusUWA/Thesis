Verbinding maken met een serverconsolesessie

Alles weergeven
Belangrijk

Als u zich wilt aanmelden bij een consolesessie op een externe server waarop Windows Server wordt uitgevoerd, moet u een beheerder zijn op de Windows-computer.

  1. Open Verbinding met extern bureaublad.

  2. Typ in het vak Computer de naam of het IP-adres van de Windows-computer waarmee u verbinding wilt maken, gevolgd door /console (bijvoorbeeld FabricamBestandsserver/console).

  3. Klik op Verbinding maken.

  4. Typ uw gebruikersnaam, wachtwoord en domein (indien vereist).

Opmerking  Deze aanmeldingsmethode werkt alleen wanneer u verbinding maakt met een server waarop Windows Server wordt uitgevoerd.
