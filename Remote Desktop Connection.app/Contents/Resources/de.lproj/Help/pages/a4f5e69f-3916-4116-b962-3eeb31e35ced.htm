Anzeigeeinstellungen für die Remotedesktopverbindung

Alles anzeigen

Sie können die Anzeigeeinstellungen für jeden Windows-basierten Computer speichern, mit dem Sie eine Verbindung herstellen. Diese Einstellungen wirken sich mehrheitlich auf die Anzeige von Grafiken aus. Je nachdem, welche Optionen ausgewählt werden, kann es zu deutlichen Beeinträchtigungen der Leistung der RemotedesktopsitzungEine Verbindung zwischen einem Clientcomputer (Macintosh) und einem Windows-basierten Remotecomputer. Diese Verbindung kann innerhalb des gleichen Netzwerks oder über das Internet hergestellt werden. Sobald die Sitzung gestartet wurde, wird der Remotedesktop (des Windows-basierten Computers) als Fenster auf dem Mac-Schreibtisch angezeigt. Das Sitzungsfenster wird geschlossen, wenn Sie die Verbindung trennen oder sich bei dem Windows-basierten Computer abmelden. kommen. Die Standardeinstellungen bieten eine optimale Leistung auf der Basis einer Breitbandverbindung.

Einstellung

Standardeinstellung

Beschreibung

Remotedesktopgröße

1024 x 768

Ändert die Bildschirmauflösung und die Größe des Fensters, in dem die Remotedesktopsitzung dargestellt wird. Wenn Sie die Sitzung in einem kleineren Fenster anzeigen möchten, können Sie für diese Option einen niedrigeren Wert festlegen.

Tipp  Wenn Sie die Größe des Fensters während einer Sitzung ändern möchten, klicken Sie im Menü Ansicht auf An Fenster anpassen, und ziehen Sie das Fenster mit dem Steuerelement auf die gewünschte Größe.

Farben

Tausende

Passt die Farbtiefe für die Sitzung an. Je nach den Netzwerkrichtlinien und der Windows-Version, die auf dem Remotecomputer ausgeführt wird, ist die maximale Farbtiefe für die Remoteverbindung möglicherweise begrenzt. Wenn Sie eine Farbtiefe auswählen, die die maximal zulässige Einstellung überschreitet, wird die maximal zulässige Farbtiefe für die Remotedesktopverbindung verwendet.

Remotedesktopfenster anzeigen

Hauptmonitor

Öffnet die Remotedesktopverbindung auf dem angegebenen Monitor. Wenn Sie mehr als einen Monitor für Ihren Macintosh verwenden, können Sie den Monitor auswählen, auf dem das Sitzungsfenster geöffnet werden soll.

Desktophintergrund anzeigen

Aus

Zeigt den benutzerdefinierten Desktophintergrund bzw. das Hintergrundbild Ihres Windows-Desktops im Sitzungsfenster an.

Schriftglättung anzeigen

Ein

Verbessert die Lesbarkeit von Bildschirmschriftarten durch Glätten von uneinheitlichen Kanten und Krümmungen.

Fensterinhalt beim Ziehen anzeigen

Aus

Zeigt den vollständigen Inhalt des Fensters beim Ziehen an, nicht nur den Rahmen.

Menü- und Fensteranimation anzeigen

Aus

Zeigt überlappende Menüs und gleitende Übergänge für Menüs und QuickInfos an.

Designs anzeigen

Ein

Zeigt benutzerdefinierte Bildschirmhintergründe, akustische Signale, Symbole und andere Elemente des Windows-basierten Computers im Sitzungsfenster an.

Bitmapzwischenspeicherung zulassen

Ein

Verbessert die Leistung durch Verringern der über die Verbindung gesendeten Informationen.

Macintosh-Menüleiste und Dock im Präsentationsmodus ausblenden

Aus

Verhindert das Anzeigen der Menüleiste und des Docks im Sitzungsfenster während der Ausführung des Präsentationsmodus. Beispielsweise können Sie diese Option auswählen, um zu verhindern, dass die Windows-Taskleiste vom Dock überlagert wird.
