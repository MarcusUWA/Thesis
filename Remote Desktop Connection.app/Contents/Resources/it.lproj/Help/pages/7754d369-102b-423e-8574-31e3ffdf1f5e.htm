Tasti di scelta rapida

Mostra tutto

È possibile eseguire rapidamente le attività utilizzando le scelte rapide da tastiera, ossia uno o più tasti che vengono premuti sulla tastiera per completare un'attività.

Le descrizioni dei tasti di scelta rapida si riferiscono al layout di tastiera degli Stati Uniti. È possibile che i tasti di altri layout non corrispondano a quelli della tastiera degli Stati Uniti. Inoltre, i tasti di scelta rapida dei computer laptop potrebbero essere diversi.

Modifica e formattazione

Quando si lavora sul desktop di Windows, è possibile eseguire le stesse attività utilizzando i tasti di scelta rapida di Windows corrispondenti. Ad esempio, per copiare del testo da un documento sul desktop di Windows, è possibile premere CTRL+C o ⌘+C. Per incollare il contenuto in un documento sul Mac, premere ⌘+V.

Utilizzo delle scelte rapide Mac sul desktop Windows

Connessione desktop remoto include assegnazioni di tasti predefinite che corrispondono ad attività sul computer Windows.

È possibile disattivare o modificare le assegnazioni di tasti seguenti. Dal menu Connessione desktop remoto scegliere Preferenze, quindi fare clic su Tastiera. Nella colonna Mac fare doppio clic sul tasto che si desidera modificare e immettere la scelta rapida desiderata.
