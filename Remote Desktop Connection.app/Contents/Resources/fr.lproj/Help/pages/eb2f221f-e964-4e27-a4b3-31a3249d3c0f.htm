Impression d'un document

Tout afficher

Vous pouvez imprimer des documents se trouvant sur votre ordinateur Windows sur une imprimante connectée à votre ordinateur Mac. Pour ce faire, commencez par partager l'imprimante avec l'ordinateur Windows.

Une fois l'imprimante disponible, elle est accessible sur l'ordinateur Windows. Si vous enregistrez le fichier de connexionFichier contenant toutes les informations nécessaires pour une connexion à distance. Par exemple, vous pouvez définir la taille d'écran pour la fenêtre de session, ou partager des lecteurs Mac avec l'ordinateur Windows en enregistrant ces préférences dans un fichier de connexion. Si vous vous connectez à plusieurs ordinateurs Windows, vous pouvez enregistrer un fichier de connexion distinct pour chacun d'eux. et vous connectez avec ce fichier de connexion lors de sessions à distance ultérieures, l'imprimante reste indisponible.

Effectuez l'une des opérations suivantes :

Mise d'une imprimante à la disposition de l'ordinateur Windows

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Cliquez sur Imprimantes, puis activez la case à cocher Utiliser une imprimante connectée au Mac.

  3. Sélectionnez l'imprimante de votre choix.

    Conseils 
     

    • Si vous voulez sélectionner une imprimante au moment de l'impression, sélectionnez Toutes les imprimantes.

    • Pour enregistrez un fichier de connexion incluant un accès à l'imprimante, dans le menu Fichier, cliquez sur Enregistrer sous, puis tapez un nom pour le fichier de connexion.

Impression de documents Windows sur une imprimante connectée à un ordinateur Mac

  1. Sur le bureau Windows, ouvrez et imprimez le document à imprimer.

  2. Sélectionnez l'imprimante sur laquelle vous voulez imprimer, puis cliquez sur OK.
