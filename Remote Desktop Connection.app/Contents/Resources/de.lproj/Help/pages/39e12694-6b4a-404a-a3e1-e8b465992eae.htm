Schließen einer Verbindung

Alles anzeigen

Sie können eine Remotesitzung schließen, indem Sie die Verbindung mit dem Windows-basierten Computer trennen oder sich bei diesem abmelden. Wenn Sie die Verbindung trennen, bleiben die verwendeten Windows-basierten Programme oder Dateien geöffnet und können später weiter verwendet werden. Wenn Sie sich bei dem Windows-basierten Computer abmelden, wird die Sitzung beendet. Alle Anwendungen in der Sitzung werden geschlossen, und Änderungen an geöffneten Dateien, die nicht gespeichert wurden, gehen verloren.

Schließen einer Verbindung mit einem Computer unter Windows 7

Mit Trennen können Sie die Verbindung trennen, ohne die Ausführung von Anwendungen auf dem Windows-basierten Computer zu unterbrechen. Mit Abmelden können Sie alle Anwendungen schließen und sich bei Windows abmelden.

Abmelden

  • Klicken Sie zuerst auf Start und dann auf Abmelden.

Trennen

  • Klicken Sie auf Start, dann auf den Pfeil neben Abmelden und dann auf Trennen.

Schließen einer Verbindung auf einem Computer unter Windows Vista

Mit Trennen können Sie die Verbindung trennen, ohne die Ausführung von Anwendungen auf dem Windows-basierten Computer zu unterbrechen. Mit Abmelden können Sie alle Anwendungen schließen und sich bei Windows abmelden.

Trennen

  • Klicken Sie auf Start und anschließend auf Trennen .

Abmelden

  • Klicken Sie auf Start und auf den Pfeil neben dem Schloss . Klicken Sie dann auf Abmelden.

Schließen einer Verbindung auf einem Computer unter Windows XP

Mit Trennen können Sie die Verbindung trennen, ohne die Ausführung von Anwendungen auf dem Windows-basierten Computer zu unterbrechen. Mit Abmelden können Sie alle Anwendungen schließen und sich bei Windows abmelden.

Trennen

  • Klicken Sie auf Start und dann auf Trennen  und anschließend erneut auf Trennen.

Abmelden

  • Klicken Sie auf Start und auf Abmelden sowie anschließend erneut auf Abmelden.

Schließen einer Verbindung auf einem Computer unter Windows Server 2003

Mit Trennen können Sie die Verbindung trennen, ohne die Ausführung von Anwendungen auf dem Windows-basierten Computer zu unterbrechen. Mit Abmelden können Sie alle Anwendungen schließen und sich bei Windows abmelden.

Trennen

  • Klicken Sie im Remotedesktop-Sitzungsfenster auf und anschließend auf Trennen.

Abmelden

  • Klicken Sie auf Start und auf Abmelden sowie anschließend erneut auf Abmelden.
