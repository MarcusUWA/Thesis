Arbeiten im Präsentationsmodus

Alles anzeigen

Wenn Sie zum ersten Mal eine Verbindung mit einem Windows-basierten Computer herstellen, wird die Remotesitzung als Fenster auf dem Schreibtisch des Macintosh angezeigt. Sie können vom Fenstermodus in den Präsentationsmodus wechseln. Dadurch wird die Anzeige maximiert, und es wird nur der Desktop des Windows-basierten Computers angezeigt.

Führen Sie eine der folgenden Aktionen aus:

Wechseln in den Präsentationsmodus

Wichtig

Wenn Sie in den Präsentationsmodus wechseln, werden die Menüleiste und das Dock des Macintosh ausgeblendet. Wenn Sie die Menüleiste oder das Dock wieder einblenden möchten, zeigen Sie auf den Rand des Bildschirms, wo sich diese normalerweise befinden. Wenn Sie zu einem späteren Zeitpunkt zum Fenstermodus zurückzukehren möchten, zeigen Sie auf den oberen Rand des Bildschirms, bis die RDC-Menüleiste angezeigt wird. Klicken Sie auf der Menüleiste auf Ansicht, und deaktivieren Sie das Kontrollkästchen Präsentationsmodus.

  • Klicken Sie im Menü Ansicht auf Vollbild.

    Tipp  Wenn Sie im Präsentationsmodus auf das Dock zeigen, überlagert das Dock die Windows-Taskleiste. Ändern Sie Ihre Anzeigeeinstellungen für die Remotedesktopverbindung, um zu verhindern, dass das Dock (und die Menüleiste) im Präsentationsmodus angezeigt werden und die Taskleiste überlagern.

Verwenden von Tastenkombinationen im Präsentationsmodus

  • Führen Sie einen der folgenden Schritte aus:

    Zweck

    Tastenkombination

    Zwischen Fenstermodus und Präsentationsmodus wechseln

    ⌘+2

    ⌘+1

    Remotesitzung im Dock ablegen

    ⌘+H

    Remotedesktopverbindung beenden
