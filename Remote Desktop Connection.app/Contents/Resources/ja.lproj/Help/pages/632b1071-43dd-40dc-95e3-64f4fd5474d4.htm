接続ファイルを作成および変更する

すべて表示

リモート デスクトップ接続 では、接続設定は既定の接続ファイルに保存されます。接続時に時間を節約するために、接続先の Windows コンピューターごとに、追加のコンピューター名や接続設定を保存することができます。たとえば、ある Windows コンピューターに接続するときはすべての Mac ディスクを共有し、別の Windows コンピューターに接続するときは 1 つの Mac フォルダーだけを共有する、というような設定の使い分けが可能になります。同じ Windows コンピューター用に複数の異なる接続ファイルを保存することもできます。

必要に応じて、次の操作を行います。

接続ファイルを作成する

  1. [RDC] メニューの [環境設定] をクリックします。

    メモ  設定を続けるために [ログイン] パネルでユーザー情報を入力する必要はありません。

  2. 接続ごとに、画面のサイズ、自動ログイン情報、セキュリティ オプションなどの接続の環境設定を選択します。

  3. [ [ファイル] ] メニューの [ [名前を付けて保存] ] をクリックします。

  4. 接続ファイルのファイル名を入力して [ [保存] ] をクリックします。

    接続が確立されると、ここで選択したオプションがすべてのリモート デスクトップ接続の既定の環境設定になります。

ヒント  Windows ベースのコンピューターにすばやく接続するには、接続ファイルをダブルクリックするか、ファイルを リモート デスクトップ接続 アイコンにドラッグします。接続ファイルは、書類/RDC Connections に保存されます。

接続ファイルを変更する

  1. リモート デスクトップ接続 で、[ [ファイル] ] メニューの [ [Edit a Connection] ] をクリックします。

  2. 目的の接続ファイルを選択し、 [開く] をクリックします。

  3. 必要な変更を加えます。

  4. [ [ファイル] ] メニューの [ [Save] ] をクリックします。

    接続が確立されると、ここで選択したオプションがすべてのリモート デスクトップ接続の既定の環境設定になります。

    ヒント  セッションの途中でも、[ [RDC] ] メニューの [ [環境設定] ] をクリックして、接続の環境設定を変更することができます。この変更は、次回の接続時に有効になります。
